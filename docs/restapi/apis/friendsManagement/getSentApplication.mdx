## 获取主动发出去的好友申请列表

</center>

### 简要描述
- 获取userID主动发起的好友申请列表，支持分页拉取。
### 请求方式
- `post` 
### 请求URL
- `http://x.x.x.x:10002/friend/get_self_friend_apply_list`

|token|eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJVc2VySUQiOiJvcGVuSU02NTQzMjEiLCJQbGF0Zm9ybUlEIjowLCJleHAiOjE2OTY0OTE4NTAsIm5iZiI6MTY4ODcxNTU1MCwiaWF0IjoxNjg4NzE1ODUwfQ.Mfhg3OiRtJY48rZbKWecNeydnHWW-GK3R2x0z8I-3Xs|必填|string|管理员token|


### 请求参数示例


```json
{
  "userID": "11111111",
  "pagination": {
    "pageNumber": 1,
    "showNumber": 100
  }
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|userID|必填|string|获取此用户主动发起的好友申请|
|pagination|必填|object|分页结构体|
|pagination.pageNumber|必填|int|当前页数，从1开始|
|pagination.showNumber|必填|int|每个分页展示申请数量|

### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|friendRequests|array|主动发出去的好友申请列表|
|friendRequests.fromUserID|string|发出申请者的用户ID|
|friendRequests.fromNickname|string|发出申请者的用户名|
|friendRequests.fromFaceURL|string|发出申请者的头像|
|friendRequests.toUserID|string|接收申请者的用户ID|
|friendRequests.toNickname|string|接收申请者的用户名|
|friendRequests.toFaceURL|string|接收申请者的头像|
|friendRequests.handleResult|string|好友申请处理结构，1为同意申请，-1为拒绝申请|
|friendRequests.reqMsg|string|申请的信息|
|friendRequests.createTime|string|创建时间|
|friendRequests.handlerUserID|string|处理者的ID|
|friendRequests.handleMsg|string|处理信息|
|friendRequests.handleTime|string|处理时间|
|friendRequests.ex|string|扩展字段|
|total|string|主动发出去的好友申请总数|
### 失败返回示例


```json
{
  "errCode": 1004,
  "errMsg": "RecordNotFoundError",
  "errDlt": "[1004]RecordNotFoundError"
}
```
### 失败返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,具体查看全局错误码文档|
|errMsg|string|错误简要信息|
|errDlt|errDlt|错误详细信息|
