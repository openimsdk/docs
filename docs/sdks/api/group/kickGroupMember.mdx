# kickGroupMember

## 功能介绍

:::info 说明

把群成员从群组中移除。

:::

:::caution 注意

（1）群主和群管理员可移除普通成员，群主可移除管理员和普通成员。 
  
  
**相关回调**:    
[onJoinedGroupDeleted](../../callback/onJoinedGroupDeleted)   
[onGroupMemberDeleted](../../callback/onGroupMemberDeleted) 


:::
