# UserIDResult

## 功能介绍

:::info
检查是否是好友的返回参数
:::

#### UserIDResult

| 字段名称 | 字段类型 | 描述                         |
| -------- | -------- | ---------------------------- |
| userID   | string   | 用户 userID                  |
| result   | int      | 1 表示好友（并且不是黑名单），0代表非好友 |
