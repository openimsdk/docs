# GroupListener

## Module Overview

:::info

Group listener, called back when group and group member info changes.

:::

| Module Name                                                                        | Description       |
| ------------------------------------------------------------------------------- | ------------------ |
| [onGroupMemberAdded](/callback/onGroupMemberAdded.md)                 | New member joined group     |
| [onGroupMemberDeleted](/callback/onGroupMemberDeleted.md)             | Member left group       |
| [onGroupMemberInfoChanged](/callback/onGroupMemberInfoChanged.md)     | Notification of member info changed |
| [onJoinedGroupAdded](/callback/onJoinedGroupAdded.md)                 | Group member changed         |
| [onJoinedGroupDeleted](/callback/onJoinedGroupDeleted.md)             | Group member changed         |
| [onGroupInfoChanged](/callback/onGroupInfoChanged.md)                 | Group info changed         |
| [onGroupApplicationAccepted](/callback/onGroupApplicationAccepted.md) | Notification of group application accepted   |
| [onGroupApplicationAdded](/callback/onGroupApplicationAdded.md)       | Notification of new group application     |
| [onGroupApplicationDeleted](/callback/onGroupApplicationDeleted.md)   | Notification of group application deleted     |
| [onGroupApplicationRejected](/callback/onGroupApplicationRejected.md) | Notification of group application rejected   |
| [onGroupDismissed](/callback/onGroupDismissed.md) | Notification of group dismissed   |
