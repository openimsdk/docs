# UserListener

## Module Overview

:::info

User information listener. Callbacks are triggered when user information changes.

:::

| Module Name                                                            | Brief Description               |
| ---------------------------------------------------------------------- | ------------------------------- |
| [onSelfUserInfoUpdate](/callback/onSelfUserInfoUpdate.md) | Personal info changed notification |
| [onUserStatusChanged](/callback/onUserStatusChanged.md)   | Subscribed user online status changed |
