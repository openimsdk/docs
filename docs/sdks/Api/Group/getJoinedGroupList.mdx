# getJoinedGroupList

## 功能介绍

:::info 说明

获取加入的群列表

:::

:::caution 注意

对于解散群也会获取到，需要应用通过群状态字段判断

:::

### 函数原型

```ts showLineNumbers
IMSDK.asyncApi('getJoinedGroupList', operationID: string): Promise<GroupItem[]>
```

### 输入参数

| 参数名称    | 参数类型 | 是否必填 | 描述                                                    |
| ----------- | -------- | -------- | ------------------------------------------------------- |
| operationID | string   | 是       | 操作 ID，用于定位问题，保持唯一，建议用当前时间和随机数 |

### 返回结果

> 通过`openim-uniapp-polyfill`包将函数 Promise 化，调用时需要使用`then`和`catch`判断并处理成功和失败回调。

| 参数名称        | 参数类型                                                         | 描述           |
| --------------- | ---------------------------------------------------------------- | -------------- |
| Promise.then()  | Promise<[GroupItem](docs/sdks/Class/Group/GroupInfo.mdx)[]\>     | 加入的群组列表 |
| Promise.catch() | Promise<[CatchResponse](docs/sdks/Class/Response/Response.mdx)\> | 调用失败回调   |

### 代码示例

```js showLineNumbers
import IMSDK from 'openim-uniapp-polyfill';

IMSDK.asyncApi('getJoinedGroupList', Date.now())
  .then((data) => {
    // 调用成功
  })
  .catch(({ errCode, errMsg }) => {
    // 调用失败
  });
```
