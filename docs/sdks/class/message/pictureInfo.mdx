# PictureInfo

## 功能介绍

:::info

图片信息详情。

:::



### PictureInfo

| 字段名称 | 字段类型 | 描述     |
| -------- | -------- | -------- |
| UUID     | string   | 唯一 ID  |
| Type     | string   | 图片类型     |
| Size     | int      | 大小     |
| Width    | int      | 宽度     |
| Height   | int      | 高度     |
| Url      | string   | 图片地址 |
