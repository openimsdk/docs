# LogLevel

:::info

设置 SDK 日志打印级别。

:::

## SDK 日志打印级别

| 枚举值名称 | 枚举值 | 描述                  |
| ---------- | ------ | --------------------- |
| WithSQL    | 6      | 打印 数据库sql 级别的日志 |
| Debug      | 5      | 打印 Debug 级别的日志 |
| Info       | 4      | 打印 Info 级别的日志  |
| Warn       | 3      | 打印 Warn 级别的日志  |
| Error      | 2      | 打印 Error 级别的日志 |
| Panic      | 1      | 打印 Panic 级别的日志 |
| Fatal      | 0      | 打印 Fatal 级别的日志 |
