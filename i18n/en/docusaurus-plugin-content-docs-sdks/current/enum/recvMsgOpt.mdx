# RecvMsgOpt

:::info

In [User Information](/class/user/userInfo.mdx) under `globalRecvMsgOpt`, globally control whether to receive offline push notifications.  
In [Conversation](/class/conversation/conversationInfo.mdx) under `recvMsgOpt`, besides controlling whether to receive offline push for that conversation, it also controls whether the unread count of that conversation is included in the total unread count.

:::

## Conversation Message Receive Options

| Enum Value | Description                                                              |
| ---------- | -------------------------------------------------------------------------|
| 0          | Normally receive messages                                                 |
| 1          | Reserved field                                                           |
| 2          | Receive messages, but no offline push. When in conversation, this conversation's unread count is not included in the total unread count |
