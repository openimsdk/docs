# OfflinePushInfo

## 功能介绍

:::info

离线推送详情。

:::


### OfflinePushInfo

| 字段名称       | 字段类型 | 描述               |
| -------------- | -------- | ------------------ |
| Title          | string   | 推送标题           |
| Desc           | string   | 推送描述           |
| iOSPushSound   | string   | 指定ios推送声音文件的名称 |
| iOSBadgeCount  | bool     | 是否算入角标数量           |
| ex             | string   | 扩展内容           |
