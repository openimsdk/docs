getSpecifiedFriendsInfo

功能介绍​

说明

获取指定好友的昵称、头像、备注，此函数从 APP 本地获取，建议一次最大10000个。

  • iOS
  • Android
  • Flutter
  • uni-app
  • Browser/MiniProgram
  • React-Native

函数原型​

    Future<List<FullUserInfo>> getFriendsInfo({
        required List<String> userIDList,
        String? operationID,
    })

输入参数​

参数名称参数类型是否必填描述
userIDListList<String>是用户 ID 列表

返回结果​

参数名称参数类型描述
~List< FullUserInfo >成功返回

代码示例​

     List<FullUserInfo> list = await OpenIM.iMManager.friendshipManager.getFriendsInfo(userIDList: []);
     // todo

函数原型​


- (void)getSpecifiedFriendsInfo:(NSArray <NSString *> *)usersID
                       onSuccess:(nullable OIMFullUsersInfoCallback)onSuccess
                       onFailure:(nullable OIMFailureCallback)onFailure;

输入参数​

参数名称参数类型是否必填描述
usersIDNSArray <NSString *>是用户 ID 列表

返回结果​

参数名称参数类型描述
onSuccessNSArray< OIMFullUserInfo * >成功返回
onFailureOIMFailureCallback失败返回

代码示例​


[OIMManager.manager getSpecifiedFriendsInfo:@[]
                                  onSuccess:^(NSArray<OIMFullUserInfo *> * _Nullable userInfos) {
} onFailure:^(NSInteger code, NSString * _Nullable msg) {
}];

函数原型​

    public void getFriendsInfo(OnBase<List<UserInfo>> base, List<String> uidList)

输入参数​

参数名称参数类型是否必填描述
callBackOnBase<List<UserInfo>>是回调接口
uidListList<String>是用户 ID 集合

代码示例​

    OpenIMClient.getInstance().friendshipManager.getFriendsInfo(new OnBase<List<UserInfo>>() {
              @Override
              public void onError(int code, String error) {

              }

              @Override
              public void onSuccess(List<UserInfo> data) {

              }
          },uidList);

函数原型​

IMSDK.getDesignatedFriendsInfo(userIDList: string[], operationID?: string): Promise<WsResponse<FullUserItem[]>>

输入参数​

参数名称参数类型是否必填描述
userIDListstring[]是用户 ID 列表

返回结果​

参数名称参数类型描述
Promise.then()Promise<WsResponse<FullUserItem[]>>好友信息列表
Promise.catch()Promise<WsResponse>调用失败回调

代码示例​

import { getSDK } from 'open-im-sdk-wasm';
const IMSDK = getSDK();
// or
// import { OpenIMSDK } from 'open-im-sdk';
// const IMSDK = new OpenIMSDK();

const userIDList = ['userID1', 'userID2'];
IMSDK.getDesignatedFriendsInfo(userIDList)
  .then(({ data }) => {
    // 调用成功
  })
  .catch(({ errCode, errMsg }) => {
    // 调用失败
  });

函数原型​

IMSDK.asyncApi('getDesignatedFriendsInfo', operationID: string, userIDList: string[]): Promise<FullUserItem[]>

输入参数​

参数名称参数类型是否必填描述
operationIDstring是操作 ID，用于定位问题，保持唯一，建议用当前时间和随机数
userIDListstring[]是用户 ID 列表

返回结果​

通过openim-uniapp-polyfill包将函数 Promise 化，调用时需要使用then和catch判断并处理成功和失败回调。

参数名称参数类型描述
Promise.then()Promise<FullUserItem[]>好友信息列表
Promise.catch()Promise<CatchResponse>调用失败回调

代码示例​

import IMSDK from 'openim-uniapp-polyfill';

const userIDList = ['userID1', 'userID2'];
IMSDK.asyncApi('getDesignatedFriendsInfo', IMSDK.uuid(), userIDList)
  .then(({ data }) => {
    // 调用成功
  })
  .catch(({ errCode, errMsg }) => {
    // 调用失败
  });

函数原型​

OpenIMSDKRN.getSpecifiedFriendsInfo( userIDList: string[],operationID: string): Promise<FriendUserItem[]>

输入参数​

参数名称参数类型是否必填描述
operationIDstring是操作 ID，用于定位问题，保持唯一，建议用当前时间和随机数
userIDListstring[]是用户 ID 列表

返回结果​

参数名称参数类型描述
Promise.then()Promise<FriendUserItem[]>好友信息列表
Promise.catch()Promise<CatchResponse>调用失败回调

代码示例​

import OpenIMSDKRN from "open-im-sdk-rn";

try {
  const userIDList = [
    // Replace with the actual list of user IDs
    'User_ID_1',
    'User_ID_2',
  ];
  const operationID = 'OperationID_value';

  const data = await OpenIMSDKRN.getSpecifiedFriendsInfo(userIDList, operationID);

  console.log("getSpecifiedFriendsInfo: Retrieved information for specified friends with user IDs", userIDList, "and received data", data);
} catch (error) {
  console.error('Error getSpecifiedFriendsInfo:', error);
}
