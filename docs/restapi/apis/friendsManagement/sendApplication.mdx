## 用户申请加好友

</center>

### 简要描述
- 先指定用户发出好友申请,需要对方同意，即可建立双向好友。
### 请求方式
- `post` 
### 请求URL
- `http://x.x.x.x:10002/friend/add_friend`

### 请求参数示例


```json
{
  "fromUserID": "11111111",
  "toUserID": "11111113",
  "reqMsg": "hello!",
  "ex": ""
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|fromUserID|必填|string|用户ID|
|toUserID|必填|string|申请被添加的用户ID|
|reqMsg|必填|string|申请信息|
|ex|必填|string|拓展字段|
### 成功返回示例


```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": ""
}
```
### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
### 失败返回示例


```json
{
  "errCode": 1301,
  "errMsg": "CanNotAddYourselfError",
  "errDlt": "[1301]CanNotAddYourselfError"
}
```
### 失败返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,具体查看全局错误码文档|
|errMsg|string|错误简要信息|
|errDlt|errDlt|错误详细信息|
