# VideoElem

## 功能介绍

:::info

视频信息。

:::


### VideoElem

| 字段名称       | 字段类型 | 描述         |
| -------------- | -------- | ------------ |
| VideoUUID      | string   | 唯一 ID      |
| VideoPath      | string   | 视频本地路径 |
| VideoType      | string   | 视频类型     |
| VideoSize      | int      | 大小         |
| Duration       | int      | 时长         |
| VideoURL       | string   | 视频远程地址     |
| SnapshotUUID   | string   | 缩略图唯一 ID      |
| SnapshotPath   | string   | 缩略图本地路径         |
| SnapshotSize   | int      | 大小         |
| SnapshotWidth  | int      | 宽度         |
| SnapshotHeight | int      | 高度         |
| SnapshotUrl    | string   | 视频缩略图地址     |
| SnapshotType   | string   | 视频类型    |
