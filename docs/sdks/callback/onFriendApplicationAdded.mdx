# onFriendApplicationAdded

## 功能介绍

:::info

用户发起好友申请后，申请发起者和接收者都会收到此回调，接收者可以选择同意或拒绝好友申请。

:::

### 返回结果

| 名称        | 类型                                                                  | 描述         |
| ----------- | --------------------------------------------------------------------- | ------------ |
| application | [FriendApplicationInfo](docs/sdks/class/relation/friendApplication.mdx) | 好友申请 |
