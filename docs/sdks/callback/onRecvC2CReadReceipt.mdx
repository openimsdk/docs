# onRecvC2CReadReceipt

## 功能介绍

:::info

自己发出的单聊消息被对方标记为已读后，消息发送者会收到此回调。

:::

### 返回结果

| 名称 | 类型                                                             | 描述     |
| ---- | ---------------------------------------------------------------- | -------- |
| list | `[]`[ReadReceiptInfo](docs/sdks/class/message/receiptInfo.mdx) | 已读回执列表 |
