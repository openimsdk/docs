6. Configuration Item Changesâ€‹

6.1 ğŸ› ï¸ Modifying Shared Configuration Itemsâ€‹

Configuration ItemFiles to ModifyAction

6.2 ğŸ”„ Modifying Special Shared Configuration Itemsâ€‹

Special configuration items: API_OPENIM_PORT, MINIO_PORT, OPENIM_IP, GRAFANA_PORT

  1. Modify these special configuration items in the .env file.
  2. Update the settings in openim-server/config/config.yaml according to the rules:
object:
  apiURL: 'http://$OPENIM_IP:$API_OPENIM_PORT'
  minio:
    endpoint: 'http://$DOCKER_BRIDGE_GATEWAY:$MINIO_PORT'
    signEndpoint: 'http://$OPENIM_IP:$MINIO_PORT'

grafanaUrl: http://$OPENIM_IP:$GRAFANA_PORT
  1. Update the settings in chat/config/config.yaml:
If IM and Chat are not on the same machine, replace `127.0.0.1` with the IP of the IM machine:
openIMUrl: 'http://127.0.0.1:$API_OPENIM_PORT'
  1. Restart IM and Chat.

OpenIM is an open source instant messaging component built by IM technical experts. OpenIM includes IM server and client SDK, which realizes important features such as high performance, light weight, and easy expansion. By integrating OpenIM components and privatizing the deployment of the server, developers can quickly integrate instant messaging and real-time network capabilities into their own applications, and ensure the security and privacy of business data.
