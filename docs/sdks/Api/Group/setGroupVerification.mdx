<TabItem value="Web">

### 函数原型

```ts showLineNumbers
IMSDK.setGroupVerification({
  groupID: string;
  verification: GroupVerificationType;
}, operationID?: string): Promise<WsResponse>
```

### 输入参数

| 参数名称     | 参数类型                                                      | 是否必填 | 描述         |
| ------------ | ------------------------------------------------------------- | -------- | ------------ |
| groupID      | string                                                        | 是       | 群 ID        |
| verification | [GroupVerificationType](docs/sdks/Enum/GroupVerification.mdx) | 是       | 入群验证方式 |

### 返回结果

| 参数名称        | 参数类型                                                      | 描述         |
| --------------- | ------------------------------------------------------------- | ------------ |
| Promise.then()  | Promise<[WsResponse](docs/sdks/Class/Response/Response.mdx)\> | 调用成功回调 |
| Promise.catch() | Promise<[WsResponse](docs/sdks/Class/Response/Response.mdx)\> | 调用失败回调 |

### 代码示例

```js showLineNumbers
import { getSDK } from 'open-im-sdk-wasm';
import { GroupVerificationType } from 'open-im-sdk-wasm/lib/types/enum';
const IMSDK = getSDK();

IMSDK.setGroupVerification({
  groupID: '',
  verification: GroupVerificationType.AllNeed,
})
  .then(() => {
    // 调用成功
  })
  .catch(({ errCode, errMsg }) => {
    // 调用失败
  });
```

<TabItem value="Uniapp">

### 函数原型

```ts showLineNumbers
IMSDK.asyncApi('setGroupVerification', operationID: string, groupID: string, verification: GroupVerificationType): Promise<void>
```

### 输入参数

| 参数名称     | 参数类型                                                      | 是否必填 | 描述                                                    |
| ------------ | ------------------------------------------------------------- | -------- | ------------------------------------------------------- |
| operationID  | string                                                        | 是       | 操作 ID，用于定位问题，保持唯一，建议用当前时间和随机数 |
| groupID      | string                                                        | 是       | 群 ID                                                   |
| verification | [GroupVerificationType](docs/sdks/Enum/GroupVerification.mdx) | 是       | 入群验证方式                                            |

### 返回结果

> 通过`openim-uniapp-polyfill`包将函数 Promise 化，调用时需要使用`then`和`catch`判断并处理成功和失败回调。

| 参数名称        | 参数类型                                                         | 描述         |
| --------------- | ---------------------------------------------------------------- | ------------ |
| Promise.then()  | Promise<void\>                                                   | 调用成功回调 |
| Promise.catch() | Promise<[CatchResponse](docs/sdks/Class/Response/Response.mdx)\> | 调用失败回调 |

### 代码示例

```js showLineNumbers
import IMSDK, { GroupVerificationType } from 'openim-uniapp-polyfill';

IMSDK.asyncApi(
  'setGroupVerification',
  Date.now(),
  'groupID',
  GroupVerificationType.AllNeed
)
  .then(() => {
    // 调用成功
  })
  .catch(({ errCode, errMsg }) => {
    // 调用失败
  });
```
