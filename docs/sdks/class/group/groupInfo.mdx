# GroupInfo

## 功能介绍

:::info

群组信息。

:::

### GroupInfo

| 字段名称               | 字段类型                                                  | 描述                                      |
| ---------------------- | --------------------------------------------------------- | ----------------------------------------- |
| GroupID                | string                                                    | 群ID                                 |
| GroupName              | string                                                    | 群名称                                    |
| Notification           | string                                                    | 群公告                                    |
| Introduction           | string                                                    | 群简介                                    |
| FaceURL                | string                                                    | 群头像                                |
| CreateTime             | int                                                       | 群创建时间                                |
| Status                 | [GroupStatus](docs/sdks/enum/groupStatus.mdx)             | 群状态  |
| CreatorUserID          | string                                                    | 创建者ID                             |
| GroupType              | [GroupType](docs/sdks/enum/groupType.mdx)                 | 群状态                                    |
| OwnerUserID            | string                                                    | 群主 ID                               |
| MemberCount            | int                                                       | 群成员数量                                |
| Ex                     | string                                                    | 扩展字段                            |
| NeedVerification       | [GroupVerification](docs/sdks/enum/groupVerification.mdx) | 进群验证方式                              |
| LookMemberInfo         | [AllowType](docs/sdks/enum/allowType.mdx)                 | 是否允许查看成员资料 |
| ApplyMemberFriend      | [AllowType](docs/sdks/enum/allowType.mdx)                 | 是否允许通过群添加好友     |
| NotificationUpdateTime | int                                                       | 群公告更新时间                            |
| NotificationUserID     | string                                                    | 群公告发布者ID                       |
