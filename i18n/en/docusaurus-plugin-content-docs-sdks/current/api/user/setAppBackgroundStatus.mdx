# setAppBackgroundStatus

## Function Introduction

:::info Description

Set whether the application is running in the background. Call when the application switches between foreground and background status.

:::

### Function Prototype

```ts showLineNumbers
IMSDK.setAppBackgroundStatus(isBackgroud: boolean, operationID?: string): Promise<WsResponse>
```

### Input Parameters

| Parameter Name | Parameter Type | Required | Description                     |
| -------------- | -------------- | -------- | ------------------------------- |
| isBackgroud    | boolean       | Yes      | Whether the application is running in the background |

### Return Results

| --------------- | ---------------------------------------------- | ---------------- |
| Promise.then()  | Promise<[WsResponse](/class/response.mdx)\>    | Successful callback |

### Function Prototype

```ts showLineNumbers
IMSDK.asyncApi('setAppBackgroundStatus', operationID: string, isBackgroud: boolean): Promise<void>
```

### Input Parameters

| Parameter Name | Parameter Type | Required | Description                                               |
| -------------- | -------------- | -------- | --------------------------------------------------------- |
| operationID    | string        | Yes      | Operation ID, used for problem tracking. Recommended to use the current time combined with a random number |
| isBackgroud    | boolean       | Yes      | Whether the application is running in the background      |

### Return Results

> The function is Promise-ified using the `openim-uniapp-polyfill` package. When calling, use `then` and `catch` to judge and handle success and failure callbacks.

| Parameter Name  | Parameter Type                                         | Description      |
| --------------- | ------------------------------------------------------ | ---------------- |
| Promise.then()  | Promise<void\>                                         | Successful callback |
| Promise.catch() | Promise<[CatchResponse](/class/response.mdx)\>         | Failure callback   |

### Code Example

```js showLineNumbers
import IMSDK from 'openim-uniapp-polyfill';

IMSDK.asyncApi('setAppBackgroundStatus', IMSDK.uuid(), true)
  .then(() => {
    // Successful call
  })
  .catch(({ errCode, errMsg }) => {
    // Failed call
  });
```

### 输入参数

| 参数名称    | 参数类型 | 是否必填 | 描述                                                    |
| ----------- | -------- | -------- | ------------------------------------------------------- |

### 返回结果

| 参数名称        | 参数类型                                                | 描述         |
| --------------- | ------------------------------------------------------- | ------------ |
| Promise.then()  | Promise<void\>                                          | 调用成功回调 |

### 代码示例

```js showLineNumbers
import OpenIMSDKRN from "open-im-sdk-rn";
try{
  await OpenIMSDKRN.setAppBackgroundStatus( true,'operationID');

}catch(error){
  console.error("Error setAppBackgroundStatus",error);
}
```
