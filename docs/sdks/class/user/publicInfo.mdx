# PublicInfo

## 功能介绍

:::info

用户公开基本信息。

:::

### PublicUserInfo

| 字段名称 | 字段类型 | 描述                        |
| -------- | -------- | --------------------------- |
| UserID   | string   | 用户 ID |
| Nickname | string   | 昵称                        |
| FaceURL  | string   | 头像                    |
| Ex       | string   | 用户扩展字段                |
| CreateTime     | int   | 创建时间 |
