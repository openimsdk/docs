# setConversationRecvMessageOpt

## 功能介绍

:::info 说明

设置会话免打扰状态。

:::

:::caution 注意

**相关回调**：      
[onConversationChanged](docs/sdks/callback/onConversationChanged.mdx)

:::

### 函数原型

```dart showLineNumbers
 Future<dynamic> setConversationRecvMessageOpt({
    required String conversationID,
    required int status,
    String? operationID,
  })
```

### 输入参数

| 参数名称           | 参数类型      | 是否必填 | 描述                                                               |
| ------------------ | ------------- | -------- | ------------------------------------------------------------------ |
| conversationID | String | 是       | 会话 ID                                                       |
| status             | [int](docs/sdks/enum/recvMsgOpt.mdx)           | 是       | 消息免打扰状态 0：正常；1：不接受消息；2：接受在线消息不接受离线消息 |

### 返回结果

| 名称 | 类型 | 描述                 |
| ---- | ---- | -------------------- |
| ~    | ~    | 无异常抛出则操作成功 |

### 代码示例

```js showLineNumbers
import OpenIMSDKRN from "open-im-sdk-rn";

try {
  const options = {
    conversationID: 'Conversation_ID',  // Replace with the actual conversation ID
    opt: 1,                           // Replace with the actual message receive option
  };
  const operationID = 'OperationID_value';

  await OpenIMSDKRN.setConversationRecvMessageOpt(options, operationID);

  console.log("setConversationRecvMessageOpt: Receive message option set for conversation with ID", options.conversationID, "to", options.opt);
} catch (error) {
  console.error('Error setConversationRecvMessageOpt:', error);
}
```
