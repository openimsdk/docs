# RecvMsgOpt

:::info

用户离线并且接入了如：个推、极光等第三方离线推送时，可以通过设置此选项来控制离线消息是否推送。

:::

## 离线消息接收选项

| 枚举值 | 描述                   |
| ------ | ---------------------- |
| 0      | 离线时会进行 APNs 推送 |
| 1      | 离线不会有推送通知     |
| 2      | 离线不会有推送通知     |
