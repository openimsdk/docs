## 多个用户对同一会话设置字段

</center>

### 简要描述
- 多个用户对同一会话可选择性的设置字段。
### 请求方式
- `post` 
### 请求URL
- `http://x.x.x.x:10002/conversation/set_conversations`

### 请求参数示例


```json
{
  "userIDs": [
    "2712330015",
    "3437150835"
  ],
  "conversation":
    {
      "conversationID": "sg_1012596513",
      "recvMsgOpt": 0,
      "isPinned": true,
      "isPrivateChat": false,
      "ex": "",
      "burnDuration": 30
    }
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|userIDs|必填|string|要设置会话字段的用户ID|
|conversations|必填|array|会话列表|
|conversations.conversationID|必填|string|会话ID|
|conversations.recvMsgOpt|选填|number|接收消息参数，0为接收消息，1为不接收消息，2为接收消息但不提醒|
|conversations.isPinned|选填|boolean|是否置顶|
|conversations.isPrivateChat|选填|boolean|是否开启阅后即焚|
|conversations.ex|选填|string|用户使用的拓展字段|
|conversations.burnDuration|选填|number|阅后即焚消息持续时间|
### 成功返回示例


```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": ""
}
```
### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
