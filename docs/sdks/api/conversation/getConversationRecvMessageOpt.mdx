IMSDK.asyncApi('getConversationRecvMessageOpt', operationID: string, conversationIDList: string[]): Promise<ConversationItem[]>

| ------------------ | -------- | -------- | ------------------------------------------------------- |

| 参数名称        | 参数类型                                                                         | 描述         |
| --------------- | -------------------------------------------------------------------------------- | ------------ |
