# onRecvOfflineNewMessage

## 功能介绍

:::info

当应用在后台运行，接收到新消息时，会收到该回调，回调中只会携带一条消息。    
设置了批量消息监听[setBatchMsgListener](docs/sdks/api/message/setBatchMsgListener.mdx)时，此回调不会触发。

:::

### 返回结果

| 名称 | 类型                                                     | 描述       |
| ---- | -------------------------------------------------------- | ---------- |
| message  | [Message](docs/sdks/class/message/messageInfo.mdx) | 新离线消息 |
