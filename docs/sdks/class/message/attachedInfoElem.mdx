# AttachedInfoElem

## 功能介绍

:::info

消息附加信息。

:::



### AttachedInfoElem

| 字段名称 | 字段类型 | 描述 |
| ---------------- | ---------------------------------------------------------------- | ---------------------------------- |
| isPrivateChat | bool | 标识是否为阅后即焚消息，仅支持单聊 |
| burnDuration | int | 阅后即焚销毁时间 |
| hasReadTime | int | 已读时间戳 |
| Progress | [UploadProgress](docs/sdks/class/message/uploadProgress.mdx) | 已读时间戳 |
