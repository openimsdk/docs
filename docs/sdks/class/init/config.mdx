# Config

## 功能介绍

:::info

初始化 SDK 的配置信息。

:::

### InitConfig

| 字段名称            | 字段类型                            | 是否必填 | 描述                                                                 |
| ------------------- | ----------------------------------- | -------- | -------------------------------------------------------------------- |
| platformID          | [int](/docs/sdks/enum/platform.mdx) | 是       | 平台号: iOS 1, Android 2, Windows 3, OSX 4, WEB 5, 小程序 6，linux 7 |
| apiAddr             | string                              | 是       | IM api 地址，一般为`http://xxx:10002`或`https://xxx/api`             |
| wsAddr              | string                              | 是       | IM ws 地址，一般为`ws://xxx:10001`或`wss://xxx/msg_gateway`          |
| dataDir             | string                              | 是       | IM 客户端 DB 存放目录                                                |
| logLevel            | int                                 | 否       | SDK 日志打印级别                                                     |
| isLogStandardOutput | bool                                | 否       | 是否将日志打印到控制台                                               |
| logFilePath         | string                              | 否       | 本地保存日志文件路径                                                 |
| logRemainCount      | int                                 | 否       | 本地保存日志文件个数                                                 |
