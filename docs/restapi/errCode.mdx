### 全局错误码

> 错误码可分为客户端错误码(10000~20000)与服务器错误码（0~9999）

### 服务器错误码

| **错误码** | **错误解释**                                                        |
|------------|--------------------------------------------------------------------|
| 0          | 没有错误                                                           |
| 500        | 服务器内部错误，通常为内部网络错误，需要检查服务器各节点运行是否正常   |
| 1001       | 参数错误，需要检查 body 参数以及 header 参数是否正确               |
| 1002       | 权限不足，一般为 header 参数中携带 token 不正确，或者权限越级操作      |
| 1003       | 数据库主键重复                                                     |
| 1004       | 数据库记录未找到                                                   |
| 1101       | 用户 ID 不存在                                                     |
| 1102       | 用户已经注册                                                       |
| 1201       | 群不存在                                                           |
| 1202       | 群已存在                                                           |
| 1203       | 用户不在群组中                                                     |
| 1204       | 群组已解散                                                         |
| 1205       | 不支持的群类型                                                     |
| 1206       | 群申请已经被处理，不需要重复处理                                   |
| 1301       | 不能添加自己为好友                                                 |
| 1302       | 已被对方拉黑                                                       |
| 1303       | 对方不是自己的好友                                                 |
| 1304       | 已经是好友关系，不需要重复申请                                     |
| 1401       | 消息已读功能被关闭                                                 |
| 1402       | 你已被禁言，不能在群里发言                                         |
| 1403       | 群已被禁言，不能发言                                               |
| 1404       | 该消息已被撤回                                                     |
| 1405       | 授权过期                                                           |
| 1501       | token 已经过期                                                     |
| 1502       | token 无效                                                         |
| 1503       | token 格式错误                                                     |
| 1504       | token 还未生效                                                     |
| 1505       | 未知 token 错误                                                    |
| 1506       | 被踢出的 token，无效                                               |
| 1507       | token 不存在                                                       |
| 1601       | 连接数超过网关最大限制                                             |
| 1602       | 连接握手参数错误                                                   |
| 1701       | 文件上传过期                                                       |
