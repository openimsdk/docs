## 获取该群指定用户的加群申请

</center>

### 简要描述
- 获取该群指定用户的加群申请
### 请求方式
- `post` 
### 请求URL
- `http://x.x.x.x:10002/group/get_group_users_req_application_list`

### 请求参数示例


```json
{
  "groupID": "234",
  "userIDs": [
    "123"
  ]
```

|userIDs|必填|array|userIDs数组|
|userIDs|必填|string|单个userID|

### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
|total|int|加群申请总数|
|groupRequests|array|加群申请数组|
|groupRequests.userInfo|object|申请加群的用户信息|
|groupRequests.userInfo.userID|string|申请加群的用户ID|
|groupRequests.userInfo.nickname|string|申请加群的用户昵称|
|groupRequests.userInfo.faceURL|string|申请加群的用户头像|
|groupRequests.userInfo.ex|string|申请加群的用户拓展字段|
|groupRequests.groupInfo|object|群组信息|
|groupRequests.groupInfo.groupID|string|群ID|
|groupRequests.groupInfo.groupName|string|群名|
|groupRequests.groupInfo.notification|string|群公告|
|groupRequests.groupInfo.introduction|string|群介绍|
|groupRequests.groupInfo.faceURL|string|群头像|
|groupRequests.groupInfo.ownerUserID|string|群主ID|
|groupRequests.groupInfo.createTime|int|群创建的时间|
|groupRequests.groupInfo.memberCount|string|群成员数量|
|groupRequests.groupInfo.ex|string|群拓展字段|
|groupRequests.groupInfo.status|int|群状态；0、正常状态1、被封禁（暂未用）2、被解散 3、处于全体禁言状态|
|groupRequests.groupInfo.creatorUserID|string|创建者ID|
|groupRequests.groupInfo.groupType|int|群类型，当前只会为2|
|groupRequests.groupInfo.needVerification|int|进群是否需要验证；0、申请加入群需要同意，成员邀请可直接进群1、所有人进群需要验证，除了群主管理员邀请进群2、直接进群|
|groupRequests.groupInfo.lookMemberInfo|int|是否能查看其他群成员信息；0、允许查看群成员信息1、不允许查看群成员信息|
|groupRequests.groupInfo.applyMemberFriend|int|群成员是否能添加好友;0、允许从群成员处添加好友1、不允许添加|
|groupRequests.groupInfo.notificationUpdateTime|int|群公告更新时间|
|groupRequests.groupInfo.notificationUserID|string|群公告用户ID|
|groupRequests.handleResult|string|处理结果，0为未处理，1为已处理，2为已拒绝|
|groupRequests.reqMsg|string|申请加群的消息|
|groupRequests.handleMsg|string|处理的信息|
|groupRequests.reqTime|string|申请时间|
|groupRequests.handleUserID|string|处理的用户ID|
|groupRequests.handleTime|string|处理的时间|
|groupRequests.ex|string|群申请拓展字段|
|groupRequests.joinSource|string|加群来源，1为管理员邀请，2为被邀请，3为搜索加入，4为扫码加入|
|groupRequests.inviterUserID|string|邀请者的ID|
