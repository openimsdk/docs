# ConversationListener

## Module Overview

:::info

Conversation listener, called back when conversation changes and conversation sync status changes.

:::

| Module Name                                                                                    | Description               |
| ------------------------------------------------------------------------------------------- | -------------------------- |
| [onSyncServerStart](/callback/onSyncServerStart.md)                               | Sync server conversation start         |
| [onSyncServerFinish](/callback/onSyncServerFinish.md)                             | Sync server conversation finish         |
| [onSyncServerFailed](/callback/onSyncServerFailed.md)                             | Sync server conversation failed         |
| [onNewConversation](/callback/onNewConversation.md)                               | There is a new conversation                 |
| [onConversationChanged](/callback/onConversationChanged.md)                       | Key info of some conversations changed |
| [onTotalUnreadMessageCountChanged](/callback/onTotalUnreadMessageCountChanged.md) | Notification of total unread message count changed       |
| [onInputStatusChanged](/callback/onInputStatusChanged.md) | Notification of input status changed       |
