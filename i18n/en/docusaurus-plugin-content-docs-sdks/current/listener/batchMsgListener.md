# BatchMsgListener

## Module Overview

:::info

Batch message listener, called back when a new message is received. After setting, the `onRecvNewMessage` and `onRecvOfflineNewMessage` callbacks in `AdvancedMsgListener` are no longer triggered.

:::

| Module Name                                                                    | Description     |
| --------------------------------------------------------------------------- | ---------------- |
| [onRecvNewMessages](/callback/onRecvNewMessages.md)               | New messages batch callback   |
| [onRecvOfflineNewMessages](/callback/onRecvOfflineNewMessages.md) | Offline messages batch callback |
