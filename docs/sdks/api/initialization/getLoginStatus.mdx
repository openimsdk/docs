# getLoginStatus

## 功能介绍

:::info 说明

获取用户登录状态，该接口如果在 SDK 未调用初始化的状态下调用，会返回-1001(Uninitialized)

:::

:::caution 注意

注意连接状态和登录状态的区别，只能在未登录状态下调用 login

:::
