# setGroupApplyMemberFriend

## 功能介绍

:::info 说明

设置普通群成员是否可以添加其他成员为好友

:::

:::caution 注意

管理员和群主才有设置权限

:::

<TabItem value="uni-app">

### 函数原型

```ts showLineNumbers
IMSDK.asyncApi('setGroupApplyMemberFriend', operationID: string, {
  groupID: string;
  rule: number;
}): Promise<void>
```

### 输入参数

| 参数名称    | 参数类型 | 是否必填 | 描述                                                    |
| ----------- | -------- | -------- | ------------------------------------------------------- |
| operationID | string   | 是       | 操作 ID，用于定位问题，保持唯一，建议用当前时间和随机数 |
| groupID     | string   | 是       | 群 ID                                                   |
| rule        | number   | 是       | 规则： 0: 默认允许 1: 不允许                            |

### 返回结果

> 通过`openim-uniapp-polyfill`包将函数 Promise 化，调用时需要使用`then`和`catch`判断并处理成功和失败回调。

| 参数名称        | 参数类型                                                | 描述         |
| --------------- | ------------------------------------------------------- | ------------ |
| Promise.then()  | Promise<void\>                                          | 调用成功回调 |
| Promise.catch() | Promise<[CatchResponse](docs/sdks/class/response.mdx)\> | 调用失败回调 |

### 代码示例

```js showLineNumbers
import IMSDK from 'openim-uniapp-polyfill';

IMSDK.asyncApi('setGroupApplyMemberFriend', Date.now(), {
  groupID: '',
  rule: 1,
})
  .then(() => {
    // 调用成功
  })
  .catch(({ errCode, errMsg }) => {
    // 调用失败
  });
```
