# onConnectFailed

## Feature Introduction

:::info

This callback is triggered when the establishment of a WebSocket connection fails.

:::

### Return Prototype

```ts showLineNumbers

onConnectFailed(data: WSEvent): void;

```

### Call Example

```js showLineNumbers
import { getSDK, CbEvents } from 'open-im-sdk-wasm';
// or
// import { OpenIMSDK, CbEvents } from 'open-im-sdk';
// const IMSDK = new OpenIMSDK();
const IMSDK = getSDK();

IMSDK.on(CbEvents.OnConnectFailed, ({ errCode, errMsg }) => {});
```

| ---- | --------------------------------------- | ------------------ |
| data | [WSEvent](/class/response.mdx) | Connection failure error information |
