# resetConversationGroupAtType

## Feature Introduction

:::info Description

Reset the @ flag to its default value.

:::

:::caution Note

When someone in the group chat uses the @ mention, or when the group owner sends a group announcement, the `groupAtType` in the conversation is triggered to the corresponding type.
After the user clicks on the conversation, calling this interface will reset the `groupAtType` in the conversation to its default value 0.

**Related Callback**:      
[onConversationChanged](docs/sdks/callback/onConversationChanged.mdx)

:::
