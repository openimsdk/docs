# UploadProgress

## 功能介绍

:::info

媒体消息上传进度信息，仅仅对于图片，视频，音频，文件等媒体消息有效，文本消息不支持。

:::

### UploadProgress

| 字段名称    | 字段类型                                                 | 描述                   |
| ----------- | -------------------------------------------------------- | ---------------------- |
| Total       | int                                                     | 消息的总字节数 |
| Current     | int                                                      | 当前已经上传的字节数    |
| UploadID    | string                                                   | 消息关联的上传id               |
