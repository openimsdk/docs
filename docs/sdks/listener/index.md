# Listener

## 模块概览

:::info

:::

| 模块名称                                                            | 模块功能简介   |
| ------------------------------------------------------------------- | -------------- |
| [ConnectListener](/listener/connectListener.md)           | 连接状态相关   |
| [UserListener](/listener/userListener.md)                 | 用户相关       |
| [FriendshipListener](/listener/friendshipListener.md)     | 关系链相关 |
| [GroupListener](/listener/groupListener.md)               | 群组相关         |
| [ConversationListener](/listener/conversationListener.md) | 会话相关       |
| [AdvancedMsgListener](/listener/advancedMsgListener.md)   | 消息相关       |
