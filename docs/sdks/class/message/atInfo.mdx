# AtInfo

## 功能介绍

:::info

@消息详细群成员信息。

:::


### AtInfo

| 字段名称      | 字段类型 | 描述                 |
| ------------- | -------- | -------------------- |
| atUserID      | String   | at 成员的 ID         |
| groupNickname | String   | at 成员的昵称/群名片 |
