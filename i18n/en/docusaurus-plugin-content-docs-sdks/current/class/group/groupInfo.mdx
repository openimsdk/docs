### GroupInfo

| 字段名称               | 字段类型                                                  | 描述                                      |
| ---------------------- | --------------------------------------------------------- | ----------------------------------------- |
| groupID                | String                                                    | 群ID                                 |
| groupName              | String                                                    | 群名称                                    |
| notification           | String                                                    | 群公告                                    |
| introduction           | String                                                    | 群简介                                    |
| faceURL                | String                                                    | 群头像                                |
| createTime             | int                                                       | 群创建时间                                |
| status                 | [int](/enum/groupStatus.mdx)                                                       | 群状态 0 正常，1 被封，2 解散，3 禁言     |
| creatorUserID          | String                                                    | 创建者ID                             |
| groupType              | [int](/enum/groupType.mdx)                 | 群状态                                    |
| ownerUserID            | String                                                    | 群主 ID                               |
| memberCount            | int                                                       | 群成员数量                                |
| ex                     | String                                                    | 扩展字段                            |
| needVerification       | [int](/enum/groupVerification.mdx) | 进群验证方式                              |
| lookMemberInfo         | [int](/enum/allowType.mdx)                 | 是否允许查看成员资料 |
| applyMemberFriend      | [int](/enum/allowType.mdx)                                                     | 是否允许通过群添加好友     |
| notificationUpdateTime | int                                                       | 群公告更新时间                            |
| notificationUserID     | String                                                    | 群公告发布者ID                       |

</TabItem>
