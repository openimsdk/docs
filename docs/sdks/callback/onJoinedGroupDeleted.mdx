# onJoinedGroupDeleted

## 功能介绍

:::info

用户所在群组的数量减少时（主动退群、群被解散等），会收到此回调。

:::

### 返回结果

| 名称 | 类型                                             | 描述     |
| ---- | ------------------------------------------------ | -------- |
| info | [GroupInfo](docs/sdks/class/group/groupInfo.mdx) | 群信息 |
