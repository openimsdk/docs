# quitGroup

## Feature Introduction

:::info Description

Leave the group.

:::

:::caution Note

(1) The group owner can leave the group only after transferring the ownership.

**Related Callbacks**:
[onJoinedGroupDeleted](../../callback/onJoinedGroupDeleted)
[onGroupMemberDeleted](../../callback/onGroupMemberDeleted)

:::
