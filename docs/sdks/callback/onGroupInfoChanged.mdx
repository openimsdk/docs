# onGroupInfoChanged

## 功能介绍

:::info

群组信息（头像、群名称等，也包括群主变化）改变时，该群所有群成员会收到此回调。

:::

### 返回结果

| 名称 | 类型                                             | 描述     |
| ---- | ------------------------------------------------ | -------- |
| info | [GroupInfo](docs/sdks/class/group/groupInfo.mdx) | 群信息 |
