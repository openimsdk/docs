## create_group

</center>

### Brief Description
- Create a group.
### Request Method
- `post` 
### Request URL
- `http://x.x.x.x:10002/group/create_group` 


### Header
| Header Name | Example Value | Optional | Type | Description |
|:----    |:-------    |:--- |---|------      |

|token|eyJhbxxxx3Xs|Required|string|Admin token|

|:----    |:-------    |:--- |
