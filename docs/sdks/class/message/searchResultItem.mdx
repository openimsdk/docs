# SearchResultItem

## 功能介绍

:::info

查找本地消息结果。

:::

### SearchResultItems

| 字段名称         | 字段类型                                                 | 描述                                                             |
| ---------------- | -------------------------------------------------------- | ---------------------------------------------------------------- |
| ConversationID   | string                                                   | 会话 ID                                                          |
| MessageCount     | int                                                      | 该会话下的消息数量                                             |
| ConversationType | [ConversationType](docs/sdks/enum/conversationType.mdx)                                                      | 会话类型  |
| ShowName         | string                                                   | 会话名称                                                           |
| FaceURL          | string                                                   | 会话头像                                                             |
| LatestMsgSendTime| int                                                   | 会话最新一条消息的发送时间                                                             |
| messageList      | `[]`[Message](docs/sdks/class/message/messageInfo.mdx) |  会话下的消息列表                                                                |
