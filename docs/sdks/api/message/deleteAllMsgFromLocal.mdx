# deleteAllMsgFromLocal

## 功能介绍

:::info 说明

删除本地所有会话中的消息，会话仍会保留。
卸载重装后还能获取到这些消息。

:::

:::caution 注意

**相关回调**：      
[onConversationChanged](docs/sdks/callback/onConversationChanged.mdx)

:::
