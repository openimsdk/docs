# BlackInfo

## 功能介绍

:::info

黑名单信息

:::

### BlackInfo

| 字段名称       | 字段类型 | 描述                    |
| -------------- | -------- | ----------------------- |
| ownerUserID    | string   | 当前登录用户 ID         |
| blockUserID    | string   | 黑名单用户 ID           |
| nickname       | string   | 用户昵称                |
| faceURL        | string   | 头像                    |
| createTime     | int      | 拉黑时间                |
| addSource      | int      | 拉黑方式                |
| operatorUserID | string   | 使之成为黑名单的用户 ID |
| attachedInfo   | string   | 暂未使用                |
| ex             | string   | 扩展字段                |
