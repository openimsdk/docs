# RelationshipListener

## Module Overview

:::info

Friend and blacklist relationship listener, called back when friend and blacklist info changes.

:::

| Module Name                                                                          | Description     |
| --------------------------------------------------------------------------------- | ---------------- |
| [onFriendApplicationAdded](/callback/onFriendApplicationAdded.md)       | Notification of new friend application |
| [onFriendApplicationRejected](/callback/onFriendApplicationRejected.md) | Friend application rejected   |
| [onFriendApplicationAccepted](/callback/onFriendApplicationAccepted.md) | Friend application accepted   |
| [onFriendApplicationDeleted](/callback/onFriendApplicationDeleted.md)   | Friend application deleted   |
| [onFriendAdded](/callback/onFriendAdded.md)                             | Notification of new friend added     |
| [onFriendDeleted](/callback/onFriendDeleted.md)                         | Notification of friend deleted     |
| [onFriendInfoChanged](/callback/onFriendInfoChanged.md)                 | Notification of friend info change |
| [onBlackAdded](/callback/onBlackAdded.md)                               | Notification of black added   |
| [onBlackDeleted](/callback/onBlackDeleted.md)                           | Notification of blacklist deleted   |
