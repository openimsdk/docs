# ConnectListener

:::info

SDK connection status listener, called back when connection status changes.

:::

| Module Name                                                        | Description                           |
| --------------------------------------------------------------- | -------------------------------------- |
| [onConnectFailed](/callback/onConnectFailed.md)       | SDK failed to connect to server                     |
| [onConnectSuccess](/callback/onConnectSuccess.md)     | SDK successfully connected to server                     |
| [onConnecting](/callback/onConnecting.md)             | SDK is connecting to server                     |
| [onKickedOffline](/callback/onKickedOffline.md)       | Account is logged in elsewhere, current device is kicked offline |
| [onUserTokenExpired](/callback/onUserTokenExpired.md) | Login token expired, please log in again             |
