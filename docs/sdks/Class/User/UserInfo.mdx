### UserInfo

| 字段名称         | 字段类型                                              | 描述                                                               |
| ---------------- | ----------------------------------------------------- | ------------------------------------------------------------------ |
| userID           | String                                                | 用户唯一 ID                                                        |
| nickName         | String                                                | 用户昵称                                                           |
| faceURL          | String                                                | 用户头像链接                                                       |
| gender           | int                                                   | 性别                                                               |
| phoneNumber      | String                                                | 手机号                                                             |
| birth            | int                                                   |                                                                    |
| email            | String                                                | 邮箱                                                               |
| ex               | String                                                | 用户扩展字段                                                       |
| remark           | String                                                | 备注                                                               |
| createTime       | int                                                   | 用户注册时间（秒）                                                 |
| appMangerLevel   | int                                                   | 用户角色 1：普通用户 2：应该管理员                                 |
| attachedInfo     | String                                                | SDK 内部扩展字段，应用层无需关注                                   |
| publicInfo       | [PublicUserInfo](docs/sdks/Class/User/PublicInfo.mdx) | 用户公开的资料                                                     |
| friendInfo       | [FriendInfo](docs/sdks/Class/Friend/FriendInfo.mdx)   | 好友才能查看的资料                                                 |
| blackInfo        | [BlacklistInfo](docs/sdks/Class/Friend/BlackInfo.mdx) | 黑名单资料                                                         |
| isFriendship     | bool                                                  | 是否好友关系                                                       |
| isBlacklist      | bool                                                  | 是否黑名单                                                         |
| globalRecvMsgOpt | int                                                   | 全局免打扰 0：正常；1：不接受消息；2：接受在线消息不接受离线消息； |
| allowAddFriend   | int                                                   | 是允许添加为好友 1：允许，2：否                                    |
| allowBeep        | int                                                   | 新消息铃声 1：允许，2：否                                          |
| allowVibration   | int                                                   | 新消息震动 1：允许，2：否                                          |
| 禁止登录         | int                                                   | 禁止登录                                                           |

</TabItem>
