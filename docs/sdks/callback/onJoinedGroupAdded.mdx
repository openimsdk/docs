# onJoinedGroupAdded

## 功能介绍

:::info

用户所在群组的数量增加时（被邀请入群、入群申请被同意等），会收到此回调。

:::


### 返回结果

| 名称 | 类型                                             | 描述     |
| ---- | ------------------------------------------------ | -------- |
| info | [GroupInfo](docs/sdks/class/group/groupInfo.mdx) | 群信息 |
