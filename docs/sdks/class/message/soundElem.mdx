# SoundElem

## 功能介绍

:::info

音频信息。

:::

### SoundElem

| 字段名称  | 字段类型 | 描述         |
| --------- | -------- | ------------ |
| UUID      | string   | 唯一 ID      |
| SoundPath | string   | 本地资源地址 |
| DataSize  | int      | 大小         |
| Duration  | int      | 时长         |
| SourceURL | string   | 语音远程地址     |
| SoundType | string   | 音频类型     |
