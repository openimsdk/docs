# onBlackDeleted

## 功能介绍

:::info

客户端调用[removeBlack](docs/sdks/api/friend/removeBlack.mdx)将某个用户移出黑名单后自己会收到该回调，回调内容为被移除的黑名单用户信息。

:::

### 调用示例

```js showLineNumbers
import { getSDK } from 'open-im-sdk-wasm';
import { CbEvents } from 'open-im-sdk-wasm/constant';
const IMSDK = getSDK();

IMSDK.on(CbEvents.OnBlackDeleted, ({ data }) => {
  // data 黑名单信息
});
```
