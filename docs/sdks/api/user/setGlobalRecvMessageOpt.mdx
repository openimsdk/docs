# setGlobalRecvMessageOpt

## 功能介绍

:::info 说明

全局设置消息提示，可用于关闭全局消息提示，包括离线推送与声音提示等。

:::

:::caution 注意

:::

### 函数原型

```dart showLineNumbers

    required int status,
    String? operationID,
```

### 输入参数

| 参数名称 | 参数类型 | 是否必填 | 描述                                                    |
| -------- | -------- | -------- | ------------------------------------------------------- |
| status   | [int](docs/sdks/enum/recvMsgOpt.mdx)      | 是       | 免打扰状态 0：正常；1：不接受消息；2：接受在线消息不接受离线消息； |

| 名称 | 类型 | 描述                 |
| ---- | ---- | -------------------- |
| ~    | ~    | 无异常抛出则操作成功 |

<TabItem value="uni-app">

### 函数原型

```ts showLineNumbers
IMSDK.asyncApi('setGlobalRecvMessageOpt', operationID: string, data: MessageReceiveOptType): Promise<void>
```

### 输入参数

| 参数名称    | 参数类型                                               | 是否必填 | 描述                                                    |
| ----------- | ------------------------------------------------------ | -------- | ------------------------------------------------------- |
| operationID | string                                                 | 是       | 操作 ID，用于定位问题，保持唯一，建议用当前时间和随机数 |
| data        | [MessageReceiveOptType](docs/sdks/enum/recvMsgOpt.mdx) | 是       | 免打扰状态                                              |

### 返回结果

> 通过`openim-uniapp-polyfill`包将函数 Promise 化，调用时需要使用`then`和`catch`判断并处理成功和失败回调。

| 参数名称        | 参数类型                                                | 描述         |
| --------------- | ------------------------------------------------------- | ------------ |
| Promise.then()  | Promise<void\>                                          | 调用成功回调 |
| Promise.catch() | Promise<[CatchResponse](docs/sdks/class/response.mdx)\> | 调用失败回调 |

### 代码示例

```js showLineNumbers
import IMSDK, { MessageReceiveOptType } from 'openim-uniapp-polyfill';

IMSDK.asyncApi(
  'setGlobalRecvMessageOpt',
  IMSDK.uuid(),
  MessageReceiveOptType.NotNotify
)
  .then(({ data }) => {
    // 调用成功
  })
  .catch(({ errCode, errMsg }) => {
    // 调用失败
  });
```
