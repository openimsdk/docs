# getSpecifiedFriendsInfo

## 功能介绍

:::info 说明

（1）获取指定好友的昵称、头像等 IM 中的资料。  
（2）建议一次最大 100 个，因为数量过多可能会导致数据包太大被后台拒绝。

:::

<TabItem value="uni-app">

### 函数原型

```ts showLineNumbers
IMSDK.asyncApi('getDesignatedFriendsInfo', operationID: string, userIDList: string[]): Promise<FriendInfo[]>
```

### 输入参数

| 参数名称    | 参数类型 | 是否必填 | 描述                                                    |
| ----------- | -------- | -------- | ------------------------------------------------------- |
| operationID | string   | 是       | 操作 ID，用于定位问题，保持唯一，建议用当前时间和随机数 |
| userIDList  | string[] | 是       | 用户 ID 列表                                            |

### 返回结果

> 通过`openim-uniapp-polyfill`包将函数 Promise 化，调用时需要使用`then`和`catch`判断并处理成功和失败回调。

| 参数名称        | 参数类型                                                         | 描述         |
| --------------- | ---------------------------------------------------------------- | ------------ |
| Promise.then()  | Promise<[FriendInfo](docs/sdks/Class/Friend/FriendInfo.mdx)[]\>  | 好友信息列表 |
| Promise.catch() | Promise<[CatchResponse](docs/sdks/Class/Response/Response.mdx)\> | 调用失败回调 |

### 代码示例

```js showLineNumbers
import IMSDK from 'openim-uniapp-polyfill';

const userIDList = ['userID1', 'userID2'];
IMSDK.asyncApi('getDesignatedFriendsInfo', Date.now(), userIDList)
  .then((data) => {
    // 调用成功
  })
  .catch(({ errCode, errMsg }) => {
    // 调用失败
  });
```
