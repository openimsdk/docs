# onGroupDismissed

## 功能介绍

:::info

群组被解散时，该群所有群成员会收到此回调。

:::

### 返回结果

| 名称      | 类型                                             | 描述     |
| --------- | ------------------------------------------------ | -------- |
| groupInfo | [GroupInfo](docs/sdks/class/group/groupInfo.mdx) | 群信息 |
