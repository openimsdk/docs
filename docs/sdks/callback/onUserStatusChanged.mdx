# onUserStatusChanged

## 功能介绍

:::info

已订阅用户的在线状态发生变化时，会触发此回调。

:::


### 返回结果

| 字段名称 | 类型                                                   | 描述     |
| ---- | ------------------------------------------------------ | -------- |
| info | [UserStatusInfo](docs/sdks/class/user/userStatusInfo.mdx) | 用户状态信息 |
