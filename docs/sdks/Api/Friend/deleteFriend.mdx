# deleteFriend

## 功能介绍

:::info 说明

删除好友

:::

:::caution 注意

（1）由于是双向好友关系，此函数仅仅是删除自己的好友列表；  
（2）发起者所有终端会收到 OnFriendDeleted 回调；  
（3）删除好友并不会自动删除会话，如要删除会话，可以再调用删除会话的函数；

:::
