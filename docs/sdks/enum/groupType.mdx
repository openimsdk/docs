:::info

群类型

## 群类型选项 {#GroupType}

| 枚举值 | 描述                 |
| ------ | -------------------- |
| 0      | 普通群（3.x 已停用） |
| 1      | 超级群（3.x 已停用） |
| 2      | 固定为2（支持大群）               |
