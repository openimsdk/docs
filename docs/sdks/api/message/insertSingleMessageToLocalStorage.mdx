# insertSingleMessageToLocalStorage

## 功能介绍

:::info 说明

插入一条群聊消息到本地，仅当前设备可见，不会同步到其他端。

:::

:::caution 注意

**相关回调**：      
[onConversationChanged](docs/sdks/callback/onConversationChanged.mdx)   
仅当前设备可见。

:::
