:::info

## 消息状态

| 枚举值 | 描述                   |
| ------ | ---------------------- |
| 1      | 发送中                 |
| 2      | 发送成功               |
| 3      | 发送失败               |
| 4      | 已删除                 |
| 5      | 已撤回，客户端不用关心 |
