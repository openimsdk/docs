# createVideoMessageByURL

## 功能介绍

:::info 说明

需要自行存储资源时，通过api上传视频和缩略图并获取下载地址后创建视频消息。

:::

:::caution 注意

通过该api创建的消息必须通过[sendMessageNotOss](docs/sdks/api/message/sendMessageNotOss.mdx)发送。

:::
