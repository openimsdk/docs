# onGroupApplicationAccepted

## Feature Introduction

:::info

When a group join request is accepted, the initiator of the request, the group owner, and the administrators of the group will receive this callback.

:::

### Call Example

```js showLineNumbers

// import { OpenIMSDK, CbEvents } from 'open-im-sdk';

  // data - Group Application
```
