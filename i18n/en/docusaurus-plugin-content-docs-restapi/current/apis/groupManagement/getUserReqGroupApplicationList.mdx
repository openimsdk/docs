## get_user_req_group_applicationList


</center>

### Brief Description
- Retrieve group join requests initiated by a specified user.
### Request Method
- `post` 
### Request URL
- `http://x.x.x.x:10002/group/get_user_req_group_applicationList` 


### Header
| Header Name | Example Value | Optional | Type | Description |
|:----    |:-------    |:--- |---|------      |
|operationID|1646445464564|Required|string|operationID used for global link tracking|
|token|eyJhbxxxx3Xs|Required|string|Admin token|


### Request Parameter Example

```json
{
  "userID": "1225441072",
  "pagination": {
    "pageNumber": 1,
    "showNumber": 100
  }
}
```
| Field Name | Optional | Type | Description |
|:----    |:-------    |:--- |---|
|userID|Required|string|User ID|
|pagination|Required|object|Pagination parameter structure|
|pagination.showNumber|Optional|int|Current page number, starting from 1|
|pagination.pageNumber|Optional|int|Number of requests on the current page|

|errCode|int|Error code, 0 indicates success|
