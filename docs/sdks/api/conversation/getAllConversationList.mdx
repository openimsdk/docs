# getAllConversationList

## 功能介绍

:::info 说明

获取所有会话列表。

:::

:::caution 注意

如果会话列表太多，建议使用[getConversationListSplit](docs/sdks/api/conversation/getConversationListSplit.mdx)分页获取。

:::
