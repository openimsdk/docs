# 三方相关

## 模块概览

:::info

三方相关（文件、日志上传，firebase的token更新等）

:::

| 模块名称                                                                  | 模块功能简介             |
| ------------------------------------------------------------------------- | ------------------------ |
| [uploadLogs](/api/third/uploadLogs.md)                 | 客户端上传日志             |
| [logs](/api/third/logs.md)                 | 打印日志到SDK中           |
| [uploadFile](/api/third/uploadFile.md)                 | 独立上传文件到服务器  |
| [updateFcmToken](/api/third/updateFcmToken.md)                 | 设置或刷新firebase用户token  |
| [setAppBadge](/api/third/setAppBadge.md)                                     | 设置App角标使用,目前仅用于firebase
