# refuseFriendApplication

## Function Introduction

:::info Description

Decline the other party's request to add you as a friend.

:::

:::caution Note

**Related Callback**:
[onFriendApplicationRejected](../../callback/onFriendApplicationRejected)

:::

| Parameter Name        | Parameter Type                                    | Description         |
| ---------------      | ------------------------------------------------- | -------------------- |

IMSDK.asyncApi('refuseFriendApplication', IMSDK.uuid(), {
  toUserID: '',
  handleMsg: '',
})
  .then(() => {
    // Successful call
  })
  .catch(({ errCode, errMsg }) => {
    // Failed call
  });

<TabItem value="React-Native">

### Function Prototype

```ts showLineNumbers
OpenIMSDKRN.refuseFriendApplication(
  userIDHandleMsg: {
    toUserID: string;
    handleMsg: string;
  },
  operationID: string
): Promise<WsResponse>;

```

### Input Parameters

| Parameter Name  | Parameter Type | Mandatory | Description             |
| --------- | -------- | -------- | ----------------- |
| toUserID  | string   | Yes       | User ID of the requester |
| handleMsg | string   | Yes       | Operation message          |

### Return Result

| Parameter Name        | Parameter Type                                                | Description        |
| --------------- | ------------------------------------------------------- | ------------ |
| Promise.then()  | Promise<void\>                                          | Callback on successful call |
| Promise.catch() | Promise<[CatchResponse](/class/response.mdx)\> | Callback on failed call |


### Code Example

```js showLineNumbers
import OpenIMSDKRN from "open-im-sdk-rn";

try {
  const userIDHandleMsg = {
    toUserID: '',
    handleMsg: '',
  };
  const operationID = 'OperationID_value';

  await OpenIMSDKRN.refuseFriendApplication(userIDHandleMsg, operationID);

  console.log("refuseFriendApplication: Refused friend application for user", userIDHandleMsg.userID, "with reason", userIDHandleMsg.handleMsg);
} catch (error) {
  console.error('Error refuseFriendApplication:', error);
}

```
