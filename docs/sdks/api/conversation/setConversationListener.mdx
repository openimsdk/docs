# setConversationListener

## 功能介绍

:::info 说明

设置会话监听器，包括新会话，会话数据改变等，异步通知到UI层。

:::

:::caution 注意

（1）在 initSDK 之后立刻调用；  
（2）只能调用一次。

:::
