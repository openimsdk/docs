```json
{
  "platformID": 2,
  "userID": "4950983283"
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|platformID|必填|string|平台ID，iOS 1, Android 2, Windows 3, OSX 4, WEB 5, 小程序 6，linux 7, AndroidPad 8, IPad 9, Admin 10|
|userID|必填|string|被踢的用户ID|
