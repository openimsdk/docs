### GroupApplicationInfo

| 字段名称      | 字段类型                                  | 描述                                           |
| ------------- | ----------------------------------------- | ---------------------------------------------- |
| groupID       | String                                    | 群唯一 ID                                      |
| groupName     | String                                    | 群名称                                         |
| notification  | String                                    | 群公告                                         |
| introduction  | String                                    | 群简介                                         |
| groupFaceURL  | String                                    | 群头像链接                                     |
| createTime    | int                                       | 申请进群时间                                   |
| status        | int                                       | 群状态：0 正常，1 被封，2 解散，3 禁言         |
| creatorUserID | String                                    | 建群者 userID                                  |
| groupType     | [GroupType](docs/sdks/Enum/GroupType.mdx) | 群类型                                         |
| ownerUserID   | String                                    | 群主 userID                                    |
| memberCount   | int                                       | 群成员数量                                     |
| userID        | String                                    | 进群申请者 userID                              |
| nickname      | String                                    | 进群申请者昵称                                 |
| userFaceURL   | String                                    | 进群申请者头像链接                             |
| handleResult  | int                                       | 处理结果：-1：拒绝，1：同意                    |
| reqMsg        | String                                    | 申请加群的附加信息                             |
| handledMsg    | String                                    | 处理附加信息                                   |
| reqTime       | int                                       | 申请进群时间                                   |
| handleUserID  | String                                    | 处理进群的 userID                              |
| handledTime   | int                                       | 处理近期申请时间                               |
| ex            | String                                    | 应用层扩展字段                                 |
| joinSource    | int                                       | 进群方式 2：通过邀请 3：通过搜索 4：通过二维码 |
| inviterUserID | String                                    | 邀请进群 userID                                |

</TabItem>
