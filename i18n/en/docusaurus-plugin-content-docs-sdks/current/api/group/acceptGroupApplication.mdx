# acceptGroupApplication

## Function Description

:::info Note

The group owner or admin approves the group joining request. After approval, the applicant will join the group.

:::

:::caution Warning

Once the request to join the group is accepted, neither the admin nor the group owner can make further operations.

**Related Callbacks**:
[onGroupApplicationAccepted](../../callback/onGroupApplicationAccepted)
[onGroupMemberAdded](../../callback/onGroupMemberAdded)
[onJoinedGroupAdded](../../callback/onJoinedGroupAdded)

:::

### 函数原型

```ts showLineNumbers
OpenIMSDKRN.acceptGroupApplication(
  {
    groupID: string,
    fromUserID: string,
    handleMsg: string,
  },
  operationID: string
);

```

### 输入参数

| 参数名称    | 参数类型 | 是否必填 | 描述                                                    |
| ----------- | -------- | -------- | ------------------------------------------------------- |
| groupID     | string   | 是       | 群ID                                                 |
| fromUserID  | string   | 是       | 申请发起者用户 ID                                       |
| handleMsg   | string   | 是       | 操作信息                                                |
| operationID | string   | 是       | 操作 ID，用于定位问题，保持唯一，建议用当前时间和随机数 |

### 返回结果

| 参数名称        | 参数类型                                                | 描述         |
| --------------- | ------------------------------------------------------- | ------------ |
| Promise.then()  | Promise<void\>                                          | 调用成功回调 |
| Promise.catch() | Promise<[CatchResponse](docs/sdks/class/response.mdx)\> | 调用失败回调 |

### 代码示例

```js showLineNumbers
import OpenIMSDKRN from "open-im-sdk-rn";

try {
  const options = {
    groupID: 'Group_ID',          // Replace with the actual group ID
    fromUserID: 'From_User_ID',  // Replace with the actual sender's user ID
    handleMsg: 'Handle_Message',  // Replace with the actual handle message
  };
  const operationID = 'OperationID_value';

  await OpenIMSDKRN.acceptGroupApplication(options, operationID);

  console.log("acceptGroupApplication: Group application accepted for group", options.groupID, "from user", options.fromUserID);
} catch (error) {
  console.error('Error acceptGroupApplication:', error);
}

```
