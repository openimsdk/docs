# onProgress

## Feature Introduction

:::info

Message sending progress callback, typically used when sending file-type messages.

:::

### Return Prototype

```dart showLineNumbers

Function(String clientMsgID, int progress)? onProgress;

```

### Return Result

| Name        | Type   | Description   |
| ----------- | ------ | ------------- |
| clientMsgID | String | Message ID    |
| progress    | int    | Progress value|

<TabItem value="Web">

### Message Sending Progress Callback

```ts showLineNumbers

onProgress(data: WSEvent<number>): void;

```

### Example Call

```js showLineNumbers
import { getSDK, CbEvents } from 'open-im-sdk-wasm';
// or
// import { OpenIMSDK, CbEvents } from 'open-im-sdk';
// const IMSDK = new OpenIMSDK();
const IMSDK = getSDK();

IMSDK.on(CbEvents.OnProgress, ({ data }) => {
  // data Message send progress
});
```
