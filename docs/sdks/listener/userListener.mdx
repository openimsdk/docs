# UserListener

## 模块概览

:::info

:::

| 模块名称                                                            | 模块功能简介               |
| ------------------------------------------------------------------- | -------------------------- |
| [onSelfInfoUpdated](docs/sdks/callback/onSelfInfoUpdated.mdx) | 个人信息变更通知           |
| [onUserStatusChanged](docs/sdks/callback/onUserStatusChanged.mdx)   | 已订阅用户在线状态变更通知 |
