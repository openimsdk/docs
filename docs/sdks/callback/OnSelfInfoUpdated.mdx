# onSelfInfoUpdated

## 功能介绍

:::info

当前登录用户个人信息改变时会收到此回调。

:::

### 返回结果

| 名称 | 类型                                          | 描述     |
| ---- | --------------------------------------------- | -------- |
| userInfo | [UserInfo](docs/sdks/class/user/userInfo.mdx) | 个人信息 |
