# onRecvOfflineNewMessages

## 功能介绍

:::info

当应用在后台运行，接收到新消息时，会收到该回调，回调中可能会携带多条消息。

:::

### 返回结果

| 名称 | 类型                                                                                              | 描述   |
| ---- | ------------------------------------------------------------------------------------------------- | ------ |
| messageList | `[]`[Message](docs/sdks/class/message/messageInfo.mdx)[]> | 新消息列表 |
