# getLoginStatus

## 功能介绍

:::info 说明

获取用户登录状态，该接口如果在sdk未调用初始化的状态下调用，会返回-1001(Uninitialized)

:::

:::caution 注意

注意连接状态和登录状态的区别，如果是未登录状态，可以调用 login，其他状态不要调用 login

:::
