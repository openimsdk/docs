# onBlackDeleted

## Feature Introduction

:::info

You will receive this callback when an entry is removed from a user's blacklist.

:::

```ts showLineNumbers

```

### Return Results

| Name | Type                                                                                           | Description        |
| ---- | ---------------------------------------------------------------------------------------------- | ------------------ |

### Call Example

```js showLineNumbers
import { getSDK } from 'open-im-sdk-wasm';
import { CbEvents } from 'open-im-sdk-wasm';
const IMSDK = getSDK();

IMSDK.on(CbEvents.OnBlackDeleted, ({ data }) => {
  // data is the blacklist information
});
```
