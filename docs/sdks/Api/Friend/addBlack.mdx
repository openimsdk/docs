# addBlack

## 功能介绍

:::info 说明

把对方拉黑

:::

:::caution 注意

意味着对方不能给你发送消息，但你可以给对方发送消息  
发起者所有终端会收到 OnBlackAdded 回调

:::

<TabItem value="uni-app">

### 函数原型

```ts showLineNumbers
IMSDK.asyncApi('addBlack', operationID: string, userID: string): Promise<void>
```

### 输入参数

| 参数名称    | 参数类型 | 是否必填 | 描述                                                    |
| ----------- | -------- | -------- | ------------------------------------------------------- |
| operationID | string   | 是       | 操作 ID，用于定位问题，保持唯一，建议用当前时间和随机数 |
| userID      | string   | 是       | 用户 ID                                                 |

### 返回结果

> 通过`openim-uniapp-polyfill`包将函数 Promise 化，调用时需要使用`then`和`catch`判断并处理成功和失败回调。

| 参数名称        | 参数类型                                                         | 描述         |
| --------------- | ---------------------------------------------------------------- | ------------ |
| Promise.then()  | Promise<void\>                                                   | 调用成功回调 |
| Promise.catch() | Promise<[CatchResponse](docs/sdks/Class/Response/Response.mdx)\> | 调用失败回调 |

### 代码示例

```js showLineNumbers
import IMSDK from 'openim-uniapp-polyfill';

IMSDK.asyncApi('addBlack', Date.now(), 'userID')
  .then(() => {
    // 调用成功
  })
  .catch(({ errCode, errMsg }) => {
    // 调用失败
  });
```
