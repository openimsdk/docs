## get_group_member_list

</center>

### Brief Description
- Retrieve the list of group members.
### Request Method
- `post` 
### Request URL
- `http://x.x.x.x:10002/group/get_group_member_list` 

### Header
| Header Name | Example Value | Optional | Type | Description |
|:----    |:-------    |:--- |---|------      |
|operationID|1646445464564|Required|string|operationID used for global link tracking|
|token|eyJhbxxxx3Xs|Required|string|Admin token|

### Request Parameter Example

```json
{
  "groupID": "2559217223",
  "pagination": {
    "pageNumber": 1,
    "showNumber": 100
  }
}
```
| Field Name | Optional | Type | Description |
|:----    |:-------    |:--- |---|
|groupID|Required|string|Group ID|
|pagination|Required|object|Structure for pagination parameters|
|pagination.pageNumber|Required|int|Current page number, starting from 1|
|pagination.showNumber|Required|int|Number of requests per page|

|:----    |:-------    |:--- |
|errCode|int|Error code, 0 indicates success|
