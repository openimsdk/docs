# onGroupApplicationRejected

## 功能介绍

:::info

入群申请被拒绝时，申请发起者以及该群的群主、管理员会收到此回调。

:::


### 返回结果

| 名称 | 类型                                                                   | 描述         |
| ---- | ---------------------------------------------------------------------- | ------------ |
| info | [GroupApplicationInfo](docs/sdks/class/group/groupApplicationInfo.mdx) | 群申请信息 |
