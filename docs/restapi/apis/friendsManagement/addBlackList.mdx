## 添加黑名单

</center>

### 简要描述
- 添加黑名单好友，黑名单好友不能向你发送消息。
### 请求方式
- `post` 
### 请求URL
- `http://x.x.x.x:10002/friend/add_black`

### 请求参数示例


```json
{
  "ownerUserID": "11111111",
  "blackUserID": "11111113"
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|ownerUserID|必填|string|用户ID|
|blackUserID|必填|string|被拉黑的用户ID|
### 成功返回示例


```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": ""
}
```

|参数名|类型|说明|

|errDlt|errDlt|错误详细信息,无错误时为空|
