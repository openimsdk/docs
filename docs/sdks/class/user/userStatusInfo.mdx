# UserOnlineStatusInfo

## 功能介绍

:::info

用户在线状态详情

:::

### UserStatusInfo

| 字段名称    | 字段类型 | 描述        |
| ----------- | -------- | ----------- |
| UserID      | string   | 用户ID |
| Status      | int      | 0 表示该用户所有平台都离线，1 表示在线        |
| PlatformIDs | `[]`[IMPlatformID](/docs/sdks/enum/platform.mdx)    | 用户在线的平台ID列表    |
