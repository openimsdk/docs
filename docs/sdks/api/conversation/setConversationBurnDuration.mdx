# setConversationBurnDuration

## 功能介绍

:::info 说明

设置单个会话上“阅后即焚”的时间。在开启阅后即焚的状态下发送的消息体上[AttachedInfoElem](docs/sdks/class/message/attachedInfoElem.mdx)的`burnDuration`会被设置为这个时间。

:::

:::caution 注意

**相关回调**：      
[onConversationChanged](docs/sdks/callback/onConversationChanged.mdx)

:::

### 代码示例

```js showLineNumbers
import OpenIMSDKRN from "open-im-sdk-rn";

try {
  const options = {
    conversationID: 'Conversation_ID',  // Replace with the actual conversation ID
    burnDuration: 60,                  // Replace with the actual burn duration in seconds
  };
  const operationID = 'OperationID_value';

  await OpenIMSDKRN.setConversationBurnDuration(options, operationID);

  console.log("setConversationBurnDuration: Burn duration set for conversation with ID", options.conversationID, "to", options.burnDuration, "seconds");
} catch (error) {
  console.error('Error setConversationBurnDuration:', error);
}
```
