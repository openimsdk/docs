### OIMFriendInfo

| 字段名称       | 字段类型  | 描述                             |
| -------------- | --------- | -------------------------------- |
| ownerUserID    | NSString  | 本人的 UserID                    |
| friendUserID   | NSString  | 好友 UserID                      |
| nickname       | NSString  | 好友的昵称                       |
| faceURL        | NSString  | 头像链接                         |
| remark         | NSString  | 好友备注                         |
| createTime     | NSInteger | 成为好友时间                     |
| addSource      | addSource | 成为好友的方式                   |
| operatorUserID | NSString  | 使双方成为好友的 UesrID          |
| attachedInfo   | NSString  | SDK 内部扩展字段，应用层无需关注 |
| ex             | NSString  | 应用层扩展字段                   |
