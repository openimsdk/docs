# NotificationElem

## 功能介绍

:::info

通知信息，例如成为好友、有人加入群、有人退出群等。
:::

:::caution 注：
其中detail为一个json字符串，需要根据json内容自行反序列化需要的内容，可以参考demo


### NotificationElem

| 字段名称 | 字段类型 | 描述 |
| -------- | -------- | ---- |
| Detail   | string   | 通知详情JSON |
