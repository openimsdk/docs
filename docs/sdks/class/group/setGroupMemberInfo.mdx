# SetGroupMemberInfo

## 功能介绍

:::info

设置群组成员信息

:::


### SetGroupMemberInfo

| 字段名称       | 字段类型                             | 描述                                               |
| -------------- | ------------------------------------ | -------------------------------------------------- |
| GroupID        | string                               | 群 ID                                              |
| UserID         | string                               | 群成员 ID                                          |
| Nickname       | string                               | 群成员昵称                                         |
| FaceURL        | string                               | 群成员头像                                         |
| RoleLevel      | [RoleLevel](docs/sdks/enum/roleLevel.mdx)                                  | 群成员角色等级          |
| Ex             | string                               | 扩展信息                                           |
