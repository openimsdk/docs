## 将用户踢出群组

</center>

### 简要描述
- 将用户踢出群组，需要群主和管理员调用。
### 请求方式
- `post` 
### 请求URL

### 请求参数示例


```json
{
  "groupID": "2137448827",
  "kickedUserIDs": [
    "2137448827"
  ],
  "reason": "nihao"
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|groupID|必填|string|群ID|
|kickedUserIDs|必填|array|被踢出群的用户ID列表|
|reason|必填|string|被踢出群的原因|
### 成功返回示例


```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": ""
}
```

|参数名|类型|说明|

|errDlt|errDlt|错误详细信息,无错误时为空|
