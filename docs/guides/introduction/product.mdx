# 🚀 OpenIMSDK

> **摆脱云锁定，自由定义通信。**
> OpenIMSDK 让每个团队都能构建属于自己的即时通讯世界。

---

## 🧭 概述

**OpenIMSDK** 是一款为开发者设计的 **应用内开源即时通讯解决方案**。
它让你能够在自己的应用中快速集成聊天、群组、好友、通知、音视频通话等功能，
无需从零搭建复杂的通信系统。

不同于 WeChat、Slack 等独立聊天应用，OpenIMSDK 专注于为应用提供 **可嵌入的通信能力**。
你完全掌控数据、交互逻辑与部署方式，可根据业务场景自由扩展与二次开发。

OpenIM 也是 **Twilio、Sendbird、Agora Chat** 等云通信平台的 **开源替代方案**，
支持私有化部署与信创环境落地，帮助团队摆脱云服务锁定。

## 🧩 OpenIMSDK 生态系统

The **OpenIM** platform consists of these core components:

- **OpenIM Server：**
开源即时通讯服务端，负责用户间的实时消息通信。
可私有部署在自有服务器上，实现对数据与系统的完全掌控。

- **OpenIM SDKs：**
提供功能完善的客户端 SDK，覆盖 iOS、Android、Web、Flutter、React Native、Electron 等平台。
轻松集成聊天、群组、好友、推送等核心功能。


---

## ☁️ 部署选项

OpenIM 提供两种灵活的部署模式，以适应不同的业务规模与控制需求。

- **🪶 OpenIM Cloud：**
由官方维护的全球分布式通信云，具备自动扩容、高可靠性与全球可用性。
非常适合需要快速上线的团队与初创项目。

- **🧱 自建私有部署：**
运行开源版本，在自有服务器或信创环境中完全掌控系统。
支持 Docker Compose、Kubernetes、一键安装脚本。
实现最大化的控制权与定制空间。

<Callout type="info">
    两种部署方案共享相同的 SDK 与 API，可无缝迁移。
</Callout>
