# deleteAllMsgFromLocalAndSvr

## Feature Introduction

:::info Description

Delete messages from all sessions locally and on the server. The sessions will still be retained.

:::

:::caution Note

**Related Callback**:
[onConversationChanged](/callback/onConversationChanged.mdx)

:::
