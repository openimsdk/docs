### 函数原型

```swift showLineNumbers

- (void)searchFriends:(OIMSearchUserParam *)searchParam
            onSuccess:(nullable OIMSearchUsersInfoCallback)onSuccess
            onFailure:(nullable OIMFailureCallback)onFailure;

```

### 输入参数

| 参数名称    | 参数类型                                                         | 是否必填 | 描述     |
| ----------- | ---------------------------------------------------------------- | -------- | -------- |
| searchParam | [OIMSearchUserParam](docs/sdks/Class/Friend/SearchUserParam.mdx) | 是       | 搜索参数 |

### 返回结果

| 参数名称  | 参数类型                                                                     | 描述     |
| --------- | ---------------------------------------------------------------------------- | -------- |
| onSuccess | NSArray< [OIMSearchUserInfo](docs/sdks/Class/Friend/SearchUserInfo.mdx) \* > | 成功返回 |
| onFailure | [OIMFailureCallback](docs/sdks/Callback/onError.mdx)                         | 失败返回 |

### 代码示例

```swift showLineNumbers

OIMSearchUserParam *param = [OIMSearchUserParam new];
param.keywordList = @[];
param.isSearchRemark = YES;
param.isSearchUserID = YES;

[OIMManager.manager searchFriends:param
                      onSuccess:^(NSArray<OIMSearchUserInfo *> * _Nullable usersInfo) {
} onFailure:^(NSInteger code, NSString * _Nullable msg) {
}];

```
