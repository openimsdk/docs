## 设置好友备注

</center>

### 简要描述
- 为ownerUserID的好友friendUserID设置备注
### 请求方式
- `post` 
### 请求URL
- `http://x.x.x.x:10002/friend/set_friend_remark`

### 请求参数示例


```json
{
  "ownerUserID": "11111111",
  "friendUserID": "11111112",
  "remark": "remark"
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|ownerUserID|必填|string|为该用户的好友设置备注|
|friendUserID|必填|string|需要备注的好友userID|
|remark|必填|string|备注|
### 成功返回示例


```json
{
  "errCode": 0,
  "errMsg": "",
  "errDlt": ""
}
```
### 成功返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,0表示成功|
|errMsg|string|错误简要信息,无错误时为空|
|errDlt|errDlt|错误详细信息,无错误时为空|
|data|object|通用数据对象，具体结构见下方|
### 失败返回示例


```json
{
  "errCode": 1004,
  "errMsg": "RecordNotFoundError",
  "errDlt": "[1004]RecordNotFoundError"
}
```
### 失败返回示例的参数说明


|参数名|类型|说明|
|:----    |:-------    |:--- |
|errCode|int|错误码,具体查看全局错误码文档|
|errMsg|string|错误简要信息|
|errDlt|errDlt|错误详细信息|
