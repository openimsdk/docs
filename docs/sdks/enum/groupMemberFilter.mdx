# GroupMemberFilter

:::info

筛选成员类型。

:::

## 筛选成员选项 {#GroupMemberFilter}

| 枚举值名称    | 枚举值 | 描述               |
| ------| ------ | ------------------ |
| GroupFilterAll           | 0      | 所有成员           |
| GroupFilterOwner         | 1      | 群主               |
| GroupFilterAdmin         | 2      |群管理员           |
| GroupFilterOrdinaryUsers        | 3      |群普通成员         |
| GroupFilterAdminAndOrdinaryUsers | 4      | 群管理员和普通成员 |
| GroupFilterOwnerAndAdmin | 5      | 群主和群管理员       |
| GroupFilterNotSelf |6   | 除开自己以外的所有成员 |
