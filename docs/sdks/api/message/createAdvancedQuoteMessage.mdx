# createAdvancedQuoteMessage

## 功能介绍

:::info 说明

创建高级引用消息，可以使得引用消息其具有富文本能力。

:::

:::caution 注意

### 函数原型

```ts showLineNumbers
IMSDK.asyncApi('createAdvancedQuoteMessage', operationID: string, {
  text: string;
  message: MessageItem;
  messageEntityList: MessageEntity[]
}): Promise<MessageItem>
```

### 输入参数

| 参数名称          | 参数类型                                                     | 是否必填 | 描述                                                    |
| ----------------- | ------------------------------------------------------------ | -------- | ------------------------------------------------------- |
| operationID       | string                                                       | 是       | 操作 ID，用于定位问题，保持唯一，建议用当前时间和随机数 |
| text              | string                                                       | 是       | 文本内容                                                |
| message           | [MessageItem](docs/sdks/class/message/messageInfo.mdx)       | 是       | 被引用消息                                              |
| messageEntityList | [MessageEntity](docs/sdks/class/message/messageEntity.mdx)[] | 是       | 自定义消息扩展信息                                      |

### 返回结果

> 通过`openim-uniapp-polyfill`包将函数 Promise 化，调用时需要使用`then`和`catch`判断并处理成功和失败回调。

| 参数名称        | 参数类型                                                         | 描述         |
| --------------- | ---------------------------------------------------------------- | ------------ |
| Promise.then()  | Promise<[MessageItem](docs/sdks/class/message/messageInfo.mdx)\> | 调用成功回调 |
| Promise.catch() | Promise<[CatchResponse](docs/sdks/class/response.mdx)\>          | 调用失败回调 |

### 代码示例

```js showLineNumbers
import IMSDK from 'openim-uniapp-polyfill';

IMSDK.asyncApi('createAdvancedQuoteMessage', IMSDK.uuid(), {
  text: '',
  message: {
    ...
  },
  messageEntityList: [
    {
      type: '',
      offset: 0,
      length: 0,
      url: "",
      onfp: ""
    }
  ]
})
  .then(({data}) => {
    // 调用成功
  })
  .catch(({ errCode, errMsg }) => {
    // 调用失败
  });
```
