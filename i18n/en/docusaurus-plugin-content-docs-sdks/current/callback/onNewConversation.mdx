# onNewConversation

## Feature Introduction

:::info

When a new conversation is initiated, this callback is received.

:::


### Return Result

| Name | Type                                                                                                             | Description          |
| ---- | ---------------------------------------------------------------------------------------------------------------- | -------------------- |
| data | [WSEvent](/class/response.mdx)<[ConversationItem](/class/conversation/conversationInfo.mdx)[]> | Conversation details |

### Example Call

```js showLineNumbers
import { getSDK } from 'open-im-sdk-wasm';
import { CbEvents } from 'open-im-sdk-wasm/constant';
const IMSDK = getSDK();

IMSDK.on(
  CbEvents.OnNewConversation,
  ({ data }: WSEvent<ConversationItem[]>) => {
    // data Conversation details
  }
);
```
