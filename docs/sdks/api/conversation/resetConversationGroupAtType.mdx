# resetConversationGroupAtType

## 功能介绍

:::info 说明

重置@标志位为默认值。

:::

:::caution 注意

当群聊中有人@，或者群主发群公告时候，会话上的 groupAtType 会触发相应类型
用户点击会话后调用此接口将会话上的 groupAtType 重置为默认值 0。   

**相关回调**：      
[onConversationChanged](docs/sdks/callback/onConversationChanged.mdx)

:::
