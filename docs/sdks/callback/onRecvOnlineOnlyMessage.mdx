# onRecvOnlineOnlyMessage

## 功能介绍

:::info

当app处于前台，接收到仅仅在线消息（该消息不会被存储，正在输入的消息不会走该回调）时，会收到该回调

:::

### 返回结果

| 名称 | 类型                                                     | 描述       |
| ---- | -------------------------------------------------------- | ---------- |
| message  | [Message](docs/sdks/class/message/messageInfo.mdx) | 在线消息 |
