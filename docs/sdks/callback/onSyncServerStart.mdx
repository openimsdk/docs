# onSyncServerStart

## 功能介绍

:::info

向服务器同步数据开始时的回调：

当reinstalled为false，代表sdk重新登录或者断线重连的数据同步，此时不会触发OnSyncServerProgress，OnSyncServerFinish会在数据同步完成后触发。

当reinstalled为true时，即表示卸载重装后的数据同步，可利用OnSyncServerProgress处理同步进度。

:::


### 返回结果

| 字段名称 | 类型                                                   | 描述     |
| ---- | ------------------------------------------------------ | -------- |
| reinstalled | bool | 是否卸载重装|
