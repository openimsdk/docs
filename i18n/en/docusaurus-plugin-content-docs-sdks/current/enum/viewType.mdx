# ViewType

:::info

When use the api of [getAdvancedHistoryMessageList](/api/message/getAdvancedHistoryMessageList) or [getAdvancedHistoryMessageListReverse](/api/message/getAdvancedHistoryMessageListReverse), used to notify the SDK of the current specific operation. 

:::

## Type of operation about get message list

| Enum Value | Description                           |
| ---------- | ------------------------------------- |
| 0          | used for getting historical message   |
| 1          | used for searching message            |
