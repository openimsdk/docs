# RelationshipListener

## 模块概览

:::info

好友及黑名单关系链监听器，当好友及黑名单信息改变时回调。

:::

| 模块名称                                                                          | 模块功能简介     |
| --------------------------------------------------------------------------------- | ---------------- |
| [onFriendApplicationAdded](/callback/onFriendApplicationAdded.md)       | 好友申请新增通知 |
| [onFriendApplicationRejected](/callback/onFriendApplicationRejected.md) | 好友申请被拒绝   |
| [onFriendApplicationAccepted](/callback/onFriendApplicationAccepted.md) | 好友申请被接受   |
| [onFriendApplicationDeleted](/callback/onFriendApplicationDeleted.md)   | 好友申请被删除   |
| [onFriendAdded](/callback/onFriendAdded.md)                             | 好友新增通知     |
| [onFriendDeleted](/callback/onFriendDeleted.md)                         | 好友删除通知     |
| [onFriendInfoChanged](/callback/onFriendInfoChanged.md)                 | 好友资料变更通知 |
| [onBlackAdded](/callback/onBlackAdded.md)                               | 黑名单新增通知   |
| [onBlackDeleted](/callback/onBlackDeleted.md)                           | 黑名单删除通知   |
