# addBlack

## Feature Introduction

:::info Explanation

Add the other party to the blacklist.

:::

:::caution Note

This means the other party cannot send you messages, but you can send messages to them.
**Relevant Callback**:
[onBlackAdded](../../callback/onBlackAdded)

:::

<TabItem value="React-Native">

### Function Prototype

```ts showLineNumbers
OpenIMSDKRN.addBlack(
  blackUserID: string,
  operationID: string
);

```

### Input Parameters

| Parameter Name    | Parameter Type | Mandatory | Description                                                   |
| ----------- | -------- | -------- | ------------------------------------------------------- |
| userID      | string   | Yes       | User ID                                                 |
| operationID | string   | Yes       | Operation ID, used for problem location, keep unique, suggest using current time and random number |

### Return Result

| Parameter Name        | Parameter Type                                                | Description        |
| --------------- | ------------------------------------------------------- | ------------ |
| Promise.then()  | Promise<void\>                                          | Callback on successful call |
| Promise.catch() | Promise<[CatchResponse](/class/response.mdx)\> | Callback on failed call |

### Code Example

```js showLineNumbers
import OpenIMSDKRN from "open-im-sdk-rn";

try {
  const blackUserID = 'Black_User_ID';  // Replace with the actual black user ID
  const operationID = 'OperationID_value';

  await OpenIMSDKRN.addBlack(blackUserID, operationID);

  console.log("addBlack: User with ID", blackUserID, "added to the black list");
} catch (error) {
  console.error('Error addBlack:', error);
}

```
