# onSyncServerFailed

## 功能介绍

:::info

由于网络或者其他原因，同步服务器数据失败时的回调。

:::

### 返回结果

| 字段名称 | 类型                                                   | 描述     |
| ---- | ------------------------------------------------------ | -------- |
| reinstalled | bool | 是否卸载重装|
