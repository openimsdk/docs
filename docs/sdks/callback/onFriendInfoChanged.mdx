# onFriendInfoChanged

## 功能介绍

:::info

好友个人信息（包括备注）改变时会收到此回调。

:::

### 返回结果

| 名称 | 类型                                                | 描述     |
| ---- | --------------------------------------------------- | -------- |
| info | [FriendInfo](docs/sdks/class/relation/friendInfo.mdx) | 好友信息 |
