# refuseGroupApplication

## 功能介绍

:::info 说明

群主和管理员拒绝进群申请

:::

:::caution 注意

进群请求被拒绝后，管理员和群主无法再操作；  
申请人以及群主和管理员会收到 onGroupApplicationRejected 回调

:::

IMSDK.asyncApi('refuseGroupApplication', operationID: string, groupID: string, fromUserID: string, handleMsg: string): Promise<void>

> 通过`openim-uniapp-polyfill`包将函数 Promise 化，调用时需要使用`then`和`catch`判断并处理成功和失败回调。
