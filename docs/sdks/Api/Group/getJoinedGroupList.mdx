# getJoinedGroupList

## 功能介绍

:::info 说明

获取加入的群列表

:::

:::caution 注意

对于解散群也会获取到，需要应用通过群状态字段判断

:::


| 参数名称  | 参数类型                                                         | 描述     |
