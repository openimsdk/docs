onGroupApplicationRejected

Feature Introduction​

info

When a group application is rejected, the applicant, as well as the group owner and administrators, receive this callback.

  • iOS
  • Android
  • Flutter
  • uni-app
  • Browser/MiniProgram

Return Prototype​

  Function(GroupApplicationInfo info)? onGroupApplicationRejected;

Return Results​

NameTypeDescription
infoGroupApplicationInfoGroup Application Info

Return Prototype​


- (void)onGroupApplicationRejected:(OIMGroupApplicationInfo *)groupApplication;

Return Results​

NameTypeDescription
applicationOIMGroupApplicationInfoGroup Application Info

Return Prototype​

void onGroupApplicationRejected(GroupApplicationInfo info);

Return Results​

NameTypeDescription
infoGroupApplicationInfoGroup Application Info

Return Prototype​


onGroupApplicationRejected(data: WSEvent<GroupApplicationItem>): void;

Return Results​

NameTypeDescription
dataWSEvent<GroupApplicationItem>Group Application Info

Call Example​

import { getSDK, CbEvents } from 'open-im-sdk-wasm';
// or
// import { OpenIMSDK, CbEvents } from 'open-im-sdk';
// const IMSDK = new OpenIMSDK();
const IMSDK = getSDK();

IMSDK.on(CbEvents.OnGroupApplicationRejected, ({ data }) => {
  // data - Group Application
});

  // data - Group Application
