### 函数原型

```dart showLineNumbers
Future updateFcmToken({
    required String fcmToken,
    required int expireTime,
    String? operationID,
  });
```

### 输入参数

| 参数名称       | 参数类型 | 是否必填 | 描述             |
| -------------- | -------- | -------- | ---------------- |
| fcmToken | String   | 是       | FCM Token         |
| expireTime | int   | 是       | FCM Token 过期时间，单位秒         |

### 返回结果

| 参数名称 | 参数类型 | 描述         |
| -------- | -------- | ------------ |
| then     | void     | 调用成功回调 |
| onError  | Function | 调用失败回调 |

```swift showLineNumbers
- (void)updateFcmToken:(NSString *)fcmToken
```

### 输入参数

| 参数名称 | 参数类型  | 是否必填 | 描述 |
| -------- | --------- | -------- | ---- |
| fcmToken    | NSString | 是       | FCM Token |
| expireTime | int   | 是       | FCM Token 过期时间，单位秒         |

### 返回结果

| 名称      | 类型                                                   | 描述     |
| --------- | ------------------------------------------------------ | -------- |
| onSuccess | OIMSuccessCallback | 成功返回 |

```swift showLineNumbers

[OIMManager.manager updateFcmToken:@""
```
