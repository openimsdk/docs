# onSyncServerProgress

## 功能介绍

:::info

向服务器同步数据的进度回调,app卸载重装时才会触发：

当app非卸载重装时，代表sdk重新登录或者断线重连的数据同步，此时不会触发OnSyncServerProgress

:::

### 返回结果

| 字段名称 | 类型                                                   | 描述     |
| ---- | ------------------------------------------------------ | -------- |
| progress  | int | 同步进度（最大100）|
