# CreateGroupReq

## 功能介绍

:::info

创建群组请求,

注：MemberUserIDs和AdminUserIDs以及OwnerUserID彼此不可重复，独立填写。

:::


### CreateGroupReq

| 字段名称               | 字段类型                                                  | 描述                                      |
| ---------------------- | --------------------------------------------------------- | ----------------------------------------- |
| MemberUserIDs          | []string   | 群成员id列表                                 |
| GroupInfo| [GroupInfo](/class/group/groupInfo.mdx)                | 群信息                                  |
| AdminUserIDs | []string              | 群管理员id列表                                    |
| OwnerUserID| string                 | 群主                                    |
