## 移除黑名单列表

</center>

### 简要描述
- 将blackUserID从ownerUserID的黑名单列表中删除，即恢复成正常关系。
### 请求方式
- `post` 
### 请求URL
- `http://x.x.x.x:10002/friend/remove_black`

### 请求参数示例


```json
{
"ownerUserID": "11111111",
"blackUserID":"11111113"
}
```
|字段名|选填|类型|说明|
|:----    |:-------    |:--- |---|
|ownerUserID|必填|string|指定的UserID|
|blackUserID|必填|string|被移除的黑名单UserID|
