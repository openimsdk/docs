# AdvancedMsgListener

## Module Overview

:::info

Message listener, called back when receiving new messages, read receipts, and revoked messages

:::

| Module Name                                                                  | Description     |
| ------------------------------------------------------------------------- | ---------------- |
| [onRecvNewMessage](/callback/onRecvNewMessage.md)               | Received new message       |
| [onRecvOfflineNewMessage](/callback/onRecvOfflineNewMessage.md) | Received offline message     |
| [onRecvC2CReadReceipt](/callback/onRecvC2CReadReceipt.md)       | C2C message read receipt |
| [onNewRecvMessageRevoked](/callback/onNewRecvMessageRevoked.md) | Callback for revoked message     |
