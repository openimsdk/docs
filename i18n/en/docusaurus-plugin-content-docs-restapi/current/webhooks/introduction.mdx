- timeout: Response timeout duration (in seconds)
- failedContinue: Whether to continue the process after timeout or API failure, generally effective only for the "before" type of callback.

## ğŸ”„ Callback Protocol

**OpenIM Server** utilizes third-party callback mechanisms based on the **HTTP/HTTPS** protocol. When initiating a callback request, **OpenIM Server** uses the **POST** request method to notify the APP's business server. The specific content of these requests will be directly included in the request body.
