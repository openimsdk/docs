# GroupMemberFilter

:::info

Filter member types.

:::

## Filter Member Options {#GroupMemberFilter}

| Enum Value | Description                |
| ---------- | -------------------------- |
| 0          | All members                |
| 1          | Group owner                |
| 2          | Group administrator        |
| 3          | Normal group member        |
| 4          | Administrator and normal member |
| 5          | Owner and administrator    |
