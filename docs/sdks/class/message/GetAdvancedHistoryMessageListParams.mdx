# GetAdvancedHistoryMessageListParams

## 功能介绍

:::info

获取历史列表参数，其中lastMinSeq参数从3.8.2+后已经移除

:::


### GetAdvancedHistoryMessageListParams 

| 字段名称       | 字段类型 | 描述         |
| -------------- | -------- | ------------ |
| ConversationID | string      | 会话 ID         |
| StartClientMsgID | string      | 从这条消息开始查询[count]条，获取的列表 index==length-1 为最新消息，所以获取下一页历史记录 startMsg=list.first       |
| Count| int      | 数量       |
