# ConversationListener

## 模块概览

:::info

会话监听器，当会话变化及会话同步状态改变时回调。

:::

| 模块名称                                                                                    | 模块功能简介               |
| ------------------------------------------------------------------------------------------- | -------------------------- |
| [onSyncServerStart](/callback/onSyncServerStart.md)                               | 同步服务器会话开始         |
| [onSyncServerFinish](/callback/onSyncServerFinish.md)                             | 同步服务器会话完成         |
| [onSyncServerFailed](/callback/onSyncServerFailed.md)                             | 同步服务器会话失败         |
| [onNewConversation](/callback/onNewConversation.md)                               | 有新的会话                 |
| [onConversationChanged](/callback/onConversationChanged.md)                       | 某些会话的关键信息发生变化 |
| [onTotalUnreadMessageCountChanged](/callback/onTotalUnreadMessageCountChanged.md) | 会话未读总数变更通知       |
| [onInputStatusChanged](/callback/onInputStatusChanged.md) | 输入状态改变通知       |
